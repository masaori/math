# 04. Yang–Lee edge singularity と非ユニタリ CFT

**問い**：ギャップ端 $\theta_e$（[02 §4](02_零点と相転移_密度と電荷アナロジー.md)）はそれ自体が臨界点だという。何が臨界で、どんな普遍類か。

**要点**：高温側で零点が届かない端 $\pm\theta_e$（複素磁場 $H=iH_e$）は、それ自体が独立した臨界点（Yang–Lee edge singularity）。密度が端で $\rho\sim|\theta-\theta_e|^\sigma$ と特異化し、その指数 $\sigma$ は**虚カップリング $\phi^3$ 場の理論**の普遍類で、2 次元では**非ユニタリ極小模型 $M(2,5)$** に対応。標準の（実磁場の）転移とは別系統の普遍性。

## 1. edge は「臨界点」

$T>T_c$ で零点はギャップ端 $\pm\theta_e$（複素磁場 $H_e=i\,\theta_e/(2\beta)$、純虚）で途切れる（[02 §4](02_零点と相転移_密度と電荷アナロジー.md)）。この端は、

- **磁化が複素磁場 $H$ の関数として分岐点をもつ点**：$H\to H_e$ で $M(H)-M(H_e)\sim(H-H_e)^{\sigma}$。
- 物理的実磁場には届かない（$H_e$ は虚）が、**自由エネルギーの解析接続の最近接特異点**として高温相の性質（帯磁率の収束半径など）を支配する。

ここでの臨界指数 $\sigma$ を **Yang–Lee edge 指数**と呼ぶ。

## 2. 密度の端指数

端近傍の零点密度：

$$
\rho(\theta)\sim|\theta-\theta_e|^{\sigma},\qquad \sigma\in(-1,0).
$$

- **平均場（$d\ge6$）**：$\sigma=-1/2$（密度が発散）。上部臨界次元は $d_c=6$。
- **$d=2$**：$\sigma=-1/6$。
- 一般に下記の場の理論・CFT で決まる。

$\sigma<0$（密度が端で発散）が edge の特徴で、通常の秩序転移（$\rho$ が消失する $a>0$、[02 §5](02_零点と相転移_密度と電荷アナロジー.md)）と**逆向きの特異性**。

## 3. 場の理論：虚カップリング $\phi^3$（Fisher 1978）

Fisher は edge singularity が**1 成分スカラー場の純虚 $\phi^3$ 相互作用**

$$
\mathcal{L}=\tfrac12(\nabla\phi)^2+i(h-h_e)\phi+ig\phi^3
$$

の IR 固定点で記述されることを示した（*PRL* **40** (1978) 1610）。$ig\phi^3$ の $i$（虚カップリング）ゆえ理論は**非ユニタリ**（ハミルトニアンが非エルミート、$\mathcal{PT}$ 対称）。上部臨界次元 $d_c=6$、$\epsilon'=6-d$ 展開で指数が計算できる。$\sigma$ は $\phi$ の異常次元 $\eta$ と

$$
\sigma=\frac{d-2+\eta}{d+2-\eta}
$$

で結ばれる（ハイパースケーリング）。

## 4. 2 次元：非ユニタリ極小模型 $M(2,5)$

$d=2$ では edge singularity は最小の**非ユニタリ共形場理論** $M(2,5)$ 極小模型（Cardy 1985, *PRL* **54** 1354）。

- 唯一の関与場のスケーリング次元 $x_\phi=-2/5$（**負**＝非ユニタリ）。
- edge 指数 $\sigma=\dfrac{x_\phi}{2-x_\phi}=\dfrac{-2/5}{2+2/5}=-\dfrac16$。
- 中心電荷 $c=-22/5$（実効中心電荷 $c_{\mathrm{eff}}=2/5$）。

これは「相転移＝CFT」の対応を**非ユニタリ**側に広げる代表例で、量子色力学の状態方程式の解析構造（QCD 臨界点探索, arXiv:2411.02663）など現代物理にも波及。

## 5. なぜ本プロジェクトに関係するか

- **位置 $\theta_e$（または $H_e$）の値**は、有限 $N$ では代数的数（最近接零点の偏角）。$N\to\infty$ の極限値は一般に超越的になりうるが、**「端が存在し、$T\to T_c^+$ で $\theta_e\to0$」という事実は可算な量化言明**（[02 §4](02_零点と相転移_密度と電荷アナロジー.md)）。
- **指数 $\sigma=-1/6$ は純粋に $\mathbb{R}/\mathbb{C}$**（連続極限・非解析性・CFT の表現論）。[本プロジェクト 05 §2](../対数順序群上の統計力学/05_相転移はΦで論じられるか.md)「臨界指数は還元不能に $\mathbb{R}/\mathbb{C}$」の典型。
- edge は「実軸の標準転移」とは別の普遍類なので、零点プログラムが**実軸の転移以外の臨界性も同じ枠**（零点軌跡の局所解析）で扱える、という普遍性の例証。

## 6. 集合帰属

| 対象 | 帰属 | 注 |
|---|---|---|
| 有限 $N$ の最近接零点（端候補） | $\overline{\mathbb{Q}}$ | 代数的 |
| 端の存在・$\theta_e\to0$ の事実 | $\mathbb{Q}$ 上の量化言明 | 可算 |
| 端位置の極限値 $\theta_e(T)$ | $\mathbb{R}$ | 連続極限。一般に超越 |
| edge 指数 $\sigma$（$-1/2,-1/6$ 等） | $\mathbb{R}$（有理数値だが $\mathbb{R}$ の臨界量） | CFT・$\epsilon$ 展開で決まる |
| $x_\phi=-2/5,\ c=-22/5$（$M(2,5)$） | $\mathbb{Q}$（CFT データ） | 値は有理数だが、意味は連続場理論 |

**注意**：$\sigma=-1/6$ や $x_\phi=-2/5$ は**値としては有理数**だが、それを与える構造（連続場理論・非解析性・極限）は $\mathbb{R}/\mathbb{C}$。値の有理性と、それが住む議論の土台は別（[本プロジェクト 07](../対数順序群上の統計力学/07_Λ帰属と可解性は別.md) の帰属と可解性の分離と同じ注意）。

## まとめ

**Yang–Lee edge $\pm\theta_e$ は純虚磁場における独立した臨界点で、密度が $\rho\sim|\theta-\theta_e|^\sigma$（$\sigma<0$）と特異化。普遍類は虚カップリング $\phi^3$ 場理論（$d_c=6$）、2 次元では非ユニタリ極小模型 $M(2,5)$（$x_\phi=-2/5$, $\sigma=-1/6$）。端の存在と閉じ方は可算な言明、指数 $\sigma$ は $\mathbb{R}/\mathbb{C}$。** 標準転移以外の臨界性も零点軌跡の局所解析で統一的に扱える例。
