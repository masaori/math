# MEMORY（Lee–Yang / Fisher 零点プログラム）

## このフォルダの目的

[対数順序群上の統計力学](../対数順序群上の統計力学/README.md) の [05](../対数順序群上の統計力学/05_相転移はΦで論じられるか.md)・[06](../対数順序群上の統計力学/06_文献調査_Ising臨界指数.md) で「相転移の足跡の実体＝Lee–Yang/Fisher 零点プログラム」と同定された対象を、独立に詳しく理解するために切り出した理解ノート群。

## 確定した骨格

- 有限系の分配関数は整係数多項式 $Z_N\in\mathbb{Z}[x]$、零点は代数的数 $\in\overline{\mathbb{Q}}$（$\mathbb{Z}[x]$ の分解体＝$\Lambda$ とは別建ての可算構造）。
- **Lee–Yang 零点**＝複素磁場／フガシティ $z=e^{-2\beta H}$。強磁性で円定理 $|z|=1$（Asano 縮約、代数的証明）。
- **Fisher 零点**＝複素温度。2D 正方 Ising で 2 円 $|z\pm1|=\sqrt2$、臨界点 $\tanh K_c=\sqrt2-1$（自己双対点、本プロジェクトの $x_c$ と一致）。
- 自由エネルギー＝零点の対数ポテンシャル、密度 $\rho=-\frac1{2\pi}\nabla^2(\beta f)$。pinch＝転移。
- Yang–Lee edge $\theta_e$：虚 $\phi^3$、2D で非ユニタリ CFT $M(2,5)$、$\sigma=-1/6$。
- FSS（IPZ）：先頭 Fisher 零点 $\sim L^{-1/\nu}$、先頭 Lee–Yang 零点 $\sim L^{-y_h}$、$y_h=(d+2-\eta)/2$。
- キュムラント法（Flindt–Garrahan）：$\kappa_n\simeq(-1)^{n-1}(n-1)!\,2\cos(n\arg s_0)/|s_0|^n$、4 連続キュムラントの線形系で先頭零点。
- 可算／$\mathbb{R}$ 境界：係数・零点・円定理・位置・転移の有無まで可算、密度・臨界指数・非解析性だけ $\mathbb{R}$。
