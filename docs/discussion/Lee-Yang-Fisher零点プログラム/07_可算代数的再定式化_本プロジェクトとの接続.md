# 07. 可算・代数的再定式化と本プロジェクトとの接続

**問い**：Lee–Yang/Fisher 零点プログラムを、本プロジェクト（対数順序群上の統計力学）の言葉でどこまで可算・代数的に書け、どこから $\mathbb{R}/\mathbb{C}$ か。本プロジェクトの差分はどこか。

**要点**：零点プログラムは大半が**可算（$\mathbb{Z}[x]$・$\overline{\mathbb{Q}}$）**で書ける。$\mathbb{R}/\mathbb{C}$ が不可避なのは「零点密度・臨界指数・非解析性」という極限の述語だけ。本プロジェクト固有の差分は、これらを**明示的に $\overline{\mathbb{Q}}$ の言明・$\mathbb{Q}$ 上の量化言明として書き、$\mathbb{R}$ 脱出を一点に隔離する**こと（[06 文献 §3](../対数順序群上の統計力学/06_文献調査_Ising臨界指数.md)）。

## 1. 可算側でどこまで言えるか

| 内容 | 可算な書き方 | 帰属 |
|---|---|---|
| 分配関数 | $Z_N(x)=\sum_m\Omega_N(m)x^m$、係数は数え上げ | $\mathbb{Z}[x]$ |
| 有限系自由エントロピー | $\Phi_N=\log Z_N(q)$（素因数分解の指数ベクトル, $q\in\mathbb{Q}_{>0}$） | $\Lambda$（[本プロジェクト 04](../対数順序群上の統計力学/04_分配関数と自由エネルギー.md)） |
| 零点 | 整係数多項式の根 | $\overline{\mathbb{Q}}$ |
| 円定理 | $z_j\bar z_j=1$（代数的等式、Asano 縮約で証明） | $\overline{\mathbb{Q}}$ 上の述語（[01](01_Lee-Yang円定理.md)） |
| 臨界点（可解模型） | 自己双対方程式 $u^2+2u-1=0$ の根 $\sqrt2-1$ | $\overline{\mathbb{Q}}$（[03](03_Fisher零点と2DIsing厳密.md)） |
| 零点軌跡 | 代数曲線 $\lvert z\pm1\rvert=\sqrt2$（$\overline{\mathbb{Q}}$ 係数） | $\overline{\mathbb{Q}}$ 係数の方程式 |
| 転移の有無 | $\forall\epsilon\in\mathbb{Q}_{>0}\,\exists N\,\exists j:\mathrm{dist}(x_j,\mathbb{R}_{>0})<\epsilon$ | $\mathbb{Q}$ 上の量化言明 |
| 先頭零点の列 | $\{x_1(L)\}_L\subset\overline{\mathbb{Q}}$ とその収束（事実） | 可算 |

**$\overline{\mathbb{Q}}$ は $\Lambda$ の拡大ではなく、$\mathbb{Z}[x]$ の分解体という別建ての可算構造**（[本プロジェクト README](../対数順序群上の統計力学/README.md)、[05 (a)](../対数順序群上の統計力学/05_相転移はΦで論じられるか.md)）。零点プログラムは $\Lambda$（対数の住処）と $\overline{\mathbb{Q}}$（フガシティ平面）の**二つの可算構造**を併走させる。

## 2. $\mathbb{R}/\mathbb{C}$ が不可避な箇所（一点に隔離）

| 内容 | なぜ $\mathbb{R}/\mathbb{C}$ か |
|---|---|
| 零点密度 $\rho=-\frac1{2\pi}\nabla^2(\beta f)$ | $N\to\infty$ の連続分布・連続体上の微分（[02](02_零点と相転移_密度と電荷アナロジー.md)） |
| pinch での非解析性 | 連続体上の解析性の否定（[本プロジェクト 05 §2](../対数順序群上の統計力学/05_相転移はΦで論じられるか.md)） |
| 臨界指数 $\nu,\eta,\sigma,\alpha$ | FSS の漸近べき・連続極限（[04](04_Lee-Yang_edge_singularityと非ユニタリCFT.md)・[05](05_有限サイズスケーリングと臨界指数抽出.md)） |
| edge の CFT $M(2,5)$ | 連続共形場理論（[04](04_Lee-Yang_edge_singularityと非ユニタリCFT.md)） |

これらはすべて**「列＋極限＋非解析性」**を要する。[本プロジェクト README 脱出 (b)](../対数順序群上の統計力学/README.md)「$N\to\infty$」に一致。**位置は代数的に決まり、非解析性そのものだけが $\mathbb{R}$**（[本プロジェクト 09 Step 4](../対数順序群上の統計力学/09_2DIsing閉形式の可算的導出.md) と同型）。

## 3. 本プロジェクトの差分（新規性の所在）

[06 文献 §3](../対数順序群上の統計力学/06_文献調査_Ising臨界指数.md) の整理通り：

1. **零点が代数的 $\in\overline{\mathbb{Q}}$ であることを明示**し、$\mathbb{C}$ 上の解析対象としてでなく**整係数多項式の根**として扱う。標準文献（Lee–Yang/Fisher/IPZ/Flindt–Garrahan）は $\mathbb{C}$ 上の解析・極限を自由に使う。
2. **転移を $\mathbb{Q}$ 上の量化言明**（$\forall\epsilon\exists N\exists j$）として書く。感受率の非有界性 $\sup_N\chi_N=\infty$ も $\forall M\in\mathbb{Q}\,\exists N$ で（[本プロジェクト 05 (b)](../対数順序群上の統計力学/05_相転移はΦで論じられるか.md)）。
3. **$\mathbb{R}$ 脱出を一点（密度・指数）に隔離**し、それ以外（係数・零点・円定理・位置・転移の有無）はすべて可算で閉じる、と明示的に台帳化する。

これらを系統的にやった文献は見当たらない（[06 文献 §3](../対数順序群上の統計力学/06_文献調査_Ising臨界指数.md)）＝**可算再定式化が未踏の差分**。

## 4. 円定理の地位（特筆）

円定理（[01](01_Lee-Yang円定理.md)）は本プロジェクトにとって特に重要：

- **証明が代数的**（Asano 縮約、解析・極限なし）で、「強磁性 Ising 分配多項式 $\in\mathbb{Z}[z]$ の根は単位円 $z\bar z=1$ 上」という**実閉体／代数の述語**。
- これは [本プロジェクト 00 §0.3](../対数順序群上の統計力学/00_記号と定義.md) の「順序判定は整数比較で無条件」と同じ精神の、**$\mathbb{R}$ を呼ばずに済む構造定理**。
- 零点プログラムの最も普遍的な部分（円定理）が、可算・代数的に完結する好例。

## 5. 未解決・進めうるスレッド

- **円定理の可算証明の精密化**：Asano 縮約を $\overline{\mathbb{Q}}$（または実閉体）上の純代数的補題として書き下す。実数解析を一切使わない版の明示。
- **転移の量化言明の厳密化**：「先頭零点が $x_c$ に詰め寄る」を、各 $L$ の代数的零点に対する**決定可能な述語の列**として定式化（Baker–Wüstholz で符号・距離が無条件判定可能か, [本プロジェクト 00 §0.3](../対数順序群上の統計力学/00_記号と定義.md)）。
- **密度・指数の $\mathbb{R}$ 性の必然性**：[本プロジェクト 05 §2](../対数順序群上の統計力学/05_相転移はΦで論じられるか.md) の「還元不能」を、零点プログラムの言葉（$\rho=-\frac1{2\pi}\nabla^2 f$ が連続ラプラシアンを要する）で具体化。
- **Fisher 零点の $\Lambda$ 表現**：有限 $L$ の $\Phi_L=\log Z_L(q)\in\Lambda$ と零点 $\in\overline{\mathbb{Q}}$ の対応を、[本プロジェクト 09](../対数順序群上の統計力学/09_2DIsing閉形式の可算的導出.md) の Step 1–4 に零点視点を重ねて記述。

## まとめ

**零点プログラムは、分配関数（$\mathbb{Z}[x]$）・零点（$\overline{\mathbb{Q}}$）・円定理（代数的述語）・臨界点の位置（$\overline{\mathbb{Q}}$）・転移の有無（$\mathbb{Q}$ 上の量化言明）まで可算で閉じ、$\mathbb{R}/\mathbb{C}$ が要るのは零点密度・臨界指数・非解析性という極限の述語だけ。本プロジェクトの差分は、この可算／$\mathbb{R}$ 境界を明示的に台帳化し、円定理のような構造定理を $\mathbb{R}$ なしで扱う点。** 本プロジェクト [05](../対数順序群上の統計力学/05_相転移はΦで論じられるか.md)・[06](../対数順序群上の統計力学/06_文献調査_Ising臨界指数.md) の主張の具体的な裏づけになる。
