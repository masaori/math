# 指数体上の統計力学

可算体 $E$（代数的数に $\exp$ を有限回適用して生成した体に $\log_E(\mathbb{N}_{>0})$ を添加したもの）の上で統計力学を定式化する試みの議論ログ。

## 立場（方針）

- $E$ のような可算集合でできるだけ議論し、不可欠な部分でだけ収束の議論をするため完備体（$\mathbb{R}/\mathbb{C}$）へ移行する。
- そのため解析（収束・微積分・実指数）は基本的に正当化されていない。具体的な値を計算するときに「外の世界」へ移行してから呼び出すツールと捉える。

## 記述ルール（このディレクトリ全体）

1. 数学的に厳密に書く。登場するすべての記号がどの集合に属するか、施す演算がどう定義されるかを明示する。特に $E$ と $\mathbb{R}/\mathbb{C}$ の分別（収束を議論できるか否か）を重視する。$\mathbb{R}/\mathbb{C}$ への移行は不可欠な連続極限に限る。微積分を不用意に正当化しない。
2. 出力はスコープを意識し過不足なく簡潔に。
