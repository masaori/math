# 指数体上の統計力学

可算な指数体 $E$ の上でできるだけ統計力学を定式化し、$\mathbb{R}/\mathbb{C}$ への移行（収束・連続極限）は不可欠な箇所に限定する、という立場での議論記録。

## 立場（二層構造）

- $E$ のような**可算集合**でできるだけ議論し、不可欠な部分でだけ収束の議論をするため完備体（$\mathbb{R}/\mathbb{C}$）に移行する。
- したがって解析（微積分・収束・指数）は基本的に**正当化されていない**。それは具体的な値を計算するときの「外の世界」に移行してからのツールと捉える。
- 核を一文で：**「数える・比を取る・$\log_E$ を自然数に当てる」だけで第〇法則と $E$ 上カノニカルまで到達でき、解析は具体値を出すとき外の世界で呼び出すツールに退避させる。**

## 記述上の原則（この議論全体で厳守）

1. **数学的厳密性**：登場するすべての記号がどの集合に属するか、施される演算がどう定義されるかをすべて明示する。とくに $E$ と $\mathbb{R}/\mathbb{C}$ の分別（収束を議論できるのか否か）を常に併記する。$E$ ないし可算集合で議論し、$\mathbb{R}/\mathbb{C}$ への移行は不可欠な連続極限に限る。微積分を不用意に正当化しない。
2. **簡潔性とスコープ**：壁打ちのため素早いイテレーションを要する。質問・依頼された内容のみを過不足なく答える。

## $E$ を出るのが不可避な三箇所

1. **$\beta$ 定数化・指数丸め**（標準カノニカル）：指数評価 $\exp_\mathbb{R}(-\beta\varepsilon)\notin E$ で離脱。
2. **示量極限** $N\to\infty$。
3. **§6 の $r\ge2$ で量子なし**のときの方向微分。

（注：「$\beta$ 定数化」自体は $E$ 内の線形化近似であり、$E$ を出ない。境界は $\beta$ 定数化と指数評価の間にある。詳細は [01](01_β定数化はEを出るか.md)。）
