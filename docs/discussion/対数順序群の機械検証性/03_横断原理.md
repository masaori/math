# 03. 横断原理：可算・代数的は検証に優しい／先行研究の空白

[01](01_自動証明_LeanCoq.md)（形式証明）と [02](02_厳密計算_SageMath.md)（厳密計算）の共通の背骨を、計算可能性・逆数学の一般論で裏付け、最後に統計力学形式化の先行研究の状況（＝空白＝新規性）を記録する。

## 1. 一般原理：離散・可算・代数的は安く、連続・実数・極限は高い

### 1.1 計算可能解析 (TTE / Weihrauch)

- 計算可能な 2 実数の**等号は決定不能**（常に停止して $x=y$ を判定するアルゴリズムは存在しない）。[X-C1]
- 順序は**半決定**のみ：$x\ne y$ なら $x<y$/$x>y$ を有限時間で確定できるが、$x=y$ では停止しない（$<$ は r.e. だが decidable でない）。[X-C2]
- **計算可能な実関数は必ず連続**。離散・可算では任意の決定可能関数が作れるのと根本的に異なる構造的制約。[X-C3]
- 実数上の計算には Type-2 Theory of Effectivity（無限ストリーム＋admissible representation）という専用枠が要る。これは実数が**非離散**で有限・可算対象のように扱えないために存在する。[X-C4]

→ [01](01_自動証明_LeanCoq.md) §2.2 の「実数の等号は構成的に決定不能・順序は半決定」、[02](02_厳密計算_SageMath.md) §2 の「`QQbar` は決定可能だが `RR` は信用できない」の理論的根拠。$\Lambda$／$\overline{\mathbb{Q}}$ は離散・可算・代数的なのでこの制約に当たらない。

### 1.2 逆数学（解析の公理コスト）

- 逆数学は定理が要する集合存在公理を測る（Big Five：RCA$_0\subset$ WKL$_0\subset$ ACA$_0\subset$ ATR$_0\subset\Pi^1_1$-CA$_0$）。[X-C5]
- **RCA$_0$**（$\Delta^0_1$ 内包＋$\Sigma^0_1$ 帰納）は**計算可能数学**を捉える基底系。可算・代数的な結果はこの低い段に住む。[X-C6]
- 解析の典型定理（Heine–Borel、最大値の存在、連続関数の性質など）は RCA$_0$ 上で WKL$_0$ や ACA$_0$ と**同値**＝より強い公理を要する。可算代数には無い公理コストが解析にはかかる、ということが定理として測られている。[X-C7]

→ 統計力学の核（有限・可算）は RCA$_0$ 級、脱出 (b) の極限・連続性（[統計力学 05](../対数順序群上の統計力学/05_相転移はΦで論じられるか.md) §2 の非解析性）は WKL$_0$/ACA$_0$ 級。$\Lambda$ に絞ることは「公理を上げない」ことに対応する。

### 1.3 形式化の実証的重量

- 証明系で代数的断片（実閉体・実代数的数）が**わざわざ分離**されているのは、決定手続きを持ち「良い算法的性質」を持つから。一般の実数はそうでない。[X-C8]
- 連続の測度・積分理論の形式化は重い（Lebesgue 積分の Coq 形式化は複数論文、中心極限定理は専用の本格形式化）。[X-C9][X-C10]
- 一般論として、実数上の基本的な不等式すら専用自動化なしには放電しづらく、決定可能断片が自動化に好まれる。[X-C11]

## 2. 三層の整合（本ノート群の核命題の再掲）

[00](00_問いと前提.md) §4 の「脱出点＝劣化点」が、§1 の一般原理で裏打ちされる：

| 層 | 核（$\Lambda$・可算・代数的） | 脱出（$\mathbb{R}$・連続・極限） |
|---|---|---|
| 数学（統計力学ノート） | 数える・比・素因数分解で閉じる | 指数評価・$N\to\infty$・微分 |
| 形式証明 | decidable、`omega`/`lia`/`decide` | `noncomputable`、解析ライブラリ |
| 厳密計算 | `ZZ`/`QQ`/`QQbar`、certified | `numerical_integral`、浮動小数点 |
| 計算可能性 | RCA$_0$、等号 decidable | WKL$_0$/ACA$_0$、等号決定不能・半決定 |

**三つの「境界」がすべて同じ場所に引かれている**——これが「$\Lambda$ に絞る」ことの工学的価値の本体。数学的厳密性の動機（[統計力学 README](../対数順序群上の統計力学/README.md)）と、機械検証しやすさの動機が、独立に同じ切り分けを支持する。

## 3. 先行研究の空白（新規性の余地）

### 3.1 統計力学・Ising の形式化はほぼ無い

- Lean/Coq/Isabelle で **2 次元 Ising 模型・分配関数・転送行列・Onsager 解を形式化した例は見当たらない**（探索範囲での否定的所見）。[X-C12]
- 最も近いのは Lean 4 の物理ライブラリ **PhysLean**（旧 HepLean）で、二準位カノニカル分布・Gibbs–Shannon エントロピーは含むが Ising/格子分配関数/Onsager は無い。[X-C13]
- 2025 年の Lean 4 による Hopfield ネット・Boltzmann マシン形式化（Gibbs サンプリング・Boltzmann 分布収束）が隣接するが、Ising/一般の分配関数は形式化していない。[X-C14]
- **Kac–Ward 行列式・Kasteleyn 向き付け・ダイマー/Pfaffian** の形式化もどの証明系にも見当たらない。[X-C15]

### 3.2 だが部品は在る

- **行列式**は Mathlib（Leibniz 和・交代多重線形）・Isabelle AFP・Coq に形式化済み。Kac–Ward／転送行列が依拠する代数基盤は存在する。[X-C16]
- 組合せ論×線形代数の形式化は若いが活発（例：Fisher の不等式の Isabelle 形式化）。[X-C17]

### 3.3 含意

- [統計力学 06](../対数順序群上の統計力学/06_文献調査_Ising臨界指数.md) §3 が「可算（$\Lambda$・$\overline{\mathbb{Q}}$）での再定式化は未踏」と述べたのと整合し、**機械検証の側でも空白**。すなわち本プロジェクトの「$\Lambda$ 上で有限格子 Ising を可算・代数的に組み、Step 1–3 を決定可能・厳密に検証する」路線は、形式証明・厳密計算の双方で先行例のない領域。
- ただし否定的所見はウェブ探索由来（リポジトリ全数監査ではない）。確度を上げるには Mathlib＋Isabelle AFP＋Coq opam の直接 grep が必要。[X-C12][X-C15]

## 4. まとめ

**「可算・離散・代数的に閉じた対象は決定可能・計算可能（RCA$_0$ 級）で、形式証明では自動化が効き厳密計算では誤差が出ない。実数・連続・極限は等号決定不能・半決定で、公理コストと形式化重量が跳ね上がる」**——この一般原理が計算可能解析と逆数学で確立しており、それが $\Lambda$ に絞る立場の工学的裏付けになる。統計力学・Ising の形式化は空白で、本プロジェクトの可算路線はそこに新規性を持つ。
