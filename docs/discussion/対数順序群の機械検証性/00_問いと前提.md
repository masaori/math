# 00. 問いと前提

## 1. 問い

[対数順序群上の統計力学](../対数順序群上の統計力学/README.md) は「核を可算な対数順序群 $\Lambda=\bigoplus_p\mathbb{Z}\ell_p$ で閉じ、$\mathbb{R}/\mathbb{C}$ への脱出を不可欠な箇所に隔離する」という**数学的厳密性**の立場だった。本ノート群はこれを**工学**の側から問い直す：

> 議論を $\Lambda$（および $\mathbb{Z},\mathbb{Q},\mathbb{Z}[x],\overline{\mathbb{Q}}$）に絞ると、
> (1) Lean/Coq/Isabelle による**形式証明**、
> (2) SageMath による**厳密数値計算**
> が、具体的にどれだけ機械にとって扱いやすくなるか。

数学的に「$\Lambda$ で閉じる」ことが、機械検証では「**決定可能（decidable）かつ実行可能（computable）な世界で閉じる**」ことに翻訳される、というのが主張の骨子。

## 2. 土台の復習（機械検証の語彙で）

| 統計力学側の対象 | 帰属 | 機械検証側の性質 |
|---|---|---|
| 多重度 $\Omega\in\mathbb{N}$ | $\mathbb{N}$ | 数え上げ。`ℕ`、決定可能等号 |
| エントロピー $S=\log\Omega$ | $\Lambda$ | 素因数分解の指数ベクトル＝`Finsupp`/$\bigoplus\mathbb{Z}$ |
| 逆温度 $\beta=S(U+1)-S(U)$ | $\Lambda$ | 群の減法＝差分商（微分でない） |
| 確率・重み $P,w$ | $\mathbb{Q}$ | 整数比、決定可能 |
| 分配多項式 $Z_L(x)$ | $\mathbb{Z}[x]$ | 整係数多項式 |
| Fisher/Lee–Yang 零点 | $\overline{\mathbb{Q}}$ | 代数的数、最小多項式＋区間 |
| 臨界点 $x_c=\sqrt2-1$ | $\overline{\mathbb{Q}}$ | 代数的、決定可能符号 |

要点は二つ。

- **$\log$ は素因数分解そのもの**：$\log:\mathbb{Q}_{>0}\xrightarrow{\sim}\Lambda$、$\log q=\sum_p v_p(q)\ell_p$。これは「実数値 $\ln$」ではなく、**整数の指数ベクトル**への写像。だから $\mathbb{R}$ も $\exp$ も収束も要らない。
- **順序は整数比較**：$\Lambda$ の元は $\log(N_+/N_-)$ の形で、$\log(N_+/N_-)>0\iff N_+>N_-$。符号・大小の判定が**2 整数の比較**に還元される。

## 3. なぜこれが機械検証で効くのか（先取り）

機械にとって「扱いやすい」とは概ね次を意味する。

- **形式証明**：命題の真偽を**有限ステップのアルゴリズム**で判定できる（`Decidable` で `decide`/`omega`/`lia` が放電できる）こと。等号・順序が決定可能な離散・代数構造はこれを満たす。
- **厳密数値計算**：値を**誤差なく**表現・比較でき、判定が**証明つき**（区間が分離するまで精度を上げる）であること。$\mathbb{Z},\mathbb{Q},\overline{\mathbb{Q}}$ はこれを満たす。

逆に $\mathbb{R}$（連続・極限・解析）は両方を破る：等号は構成的に決定不能、順序は半決定のみ、計算は浮動小数点近似に落ちる。詳細は [01](01_自動証明_LeanCoq.md)・[02](02_厳密計算_SageMath.md)、一般原理は [03](03_横断原理.md)。

## 4. 脱出点 = 劣化点の対応（本ノート群の中心命題）

統計力学ノートが特定した「$\Lambda$ を出て一気に $\mathbb{R}$ に飛ぶ箇所」は、そのまま**機械検証が劣化する箇所**に一致する。これが「$\Lambda$ に絞る」ことの工学的価値を一行で示す。

| 統計力学ノートの脱出点 | 形式証明での劣化 | 厳密計算での劣化 |
|---|---|---|
| (a) $\beta$ 定数化後の**指数評価** $e^{-\beta\varepsilon}$ | 超越数・実関数 → `noncomputable`、解析ライブラリ依存 | $\mathbb{R}$ 値、`RDF` 浮動小数点 |
| (b) **示量極限** $N\to\infty$・Onsager 積分 | 極限・連続性 → WKL$_0$/ACA$_0$ 級の重い証明 | `numerical_integral` → 誤差つき近似 |
| (c) $r\ge2$ で量子なしの**方向微分** | 微分・極限 → 解析の形式化重量 | 数値微分 → 近似 |

対して核（[統計力学 00](../対数順序群上の統計力学/00_記号と定義.md) §1–7、有限格子の $Z_L,\Phi_L,\beta$、零点、臨界点の**位置**）は、形式証明では決定可能・自動放電可能、厳密計算では誤差ゼロ・証明つき判定可能。

**つまり「数学的厳密性のために $\Lambda$ に絞る」ことと「機械検証しやすさのために決定可能・厳密な対象に絞る」ことは、同じ境界を引いている。** これが本ノート群の主張で、[01](01_自動証明_LeanCoq.md)・[02](02_厳密計算_SageMath.md)・[03](03_横断原理.md) で各論を埋める。
