# 08. Kramers–Wannier 双対性（補足）

**位置づけ**：双対性は閉形式導出のインサイトになりうるが、本筋（$\Lambda$ 上の定式化・$Z$→Massieu→零点）には必須でない。**補足情報**として、定義・可算表現・一般論・「何を買うか」を簡潔に記録する。

## 1. 定義（グラフ的・組合せ的）

平面グラフ $G=(V,E)$ の Ising 分配関数 $Z_G(K)=\sum_\sigma\prod_{(uv)\in E}e^{K\sigma_u\sigma_v}$（$K=\beta J$）。

- **高温展開**：$e^{K\sigma_u\sigma_v}=\cosh K(1+\sigma_u\sigma_v\tanh K)$ → $Z_G=(\cosh K)^{|E|}2^{|V|}\sum_{\gamma\in\mathcal{E}(G)}t^{|\gamma|}$、$t=\tanh K$、$\mathcal{E}(G)$＝偶部分グラフ。
- **低温展開**：ドメイン壁＝双対 $G^\ast$ の閉曲線 → $Z_G=2e^{K|E|}\sum_{\gamma\in\mathcal{E}(G^\ast)}(e^{-2K})^{|\gamma|}$。
- **双対性**：両展開の一致から、結合の対合
$$
e^{-2K}=\tanh K^\ast\iff \sinh2K\,\sinh2K^\ast=1,\qquad Z_G(K)=(\text{前因子})\,Z_{G^\ast}(K^\ast).
$$
正方格子は $G^\ast\cong G$（自己双対）。

## 2. 可算（$\Lambda$）表現

変数 $t=\tanh K$ で偶部分グラフ生成多項式 $P_G(t)=\sum_{\gamma\in\mathcal{E}(G)}t^{|\gamma|}\in\mathbb{Z}[t]$。双対は **$\mathbb{Q}$-メビウス変換** $t\mapsto t^\ast=\frac{1-t}{1+t}$ の下での $\mathbb{Z}[t]$ の関数等式で、$\exp$ も $\mathbb{R}$ も使わない。有理点で評価すれば対数レベルの $\Lambda$ 恒等式 $\log\hat Z_G(t)=\log(\text{有理前因子})+\log\hat Z_{G^\ast}(t^\ast)\in\Lambda$。自己双対不動点 $t^2+2t-1=0$、$t_c=\sqrt2-1\in\overline{\mathbb{Q}}$。$\Lambda$ を出るのは $t=\tanh K$ を実結合 $K$ に戻す段（$\exp$）だけで、双対自体には不要。

## 3. 一般論の抽象度（2D Ising 特有ではない）

| 抽象度 | 双対の正体 | 自己双対 |
|---|---|---|
| 平面グラフ | 平面双対 $G\leftrightarrow G^\ast$ | 正方格子のみ |
| アーベルスピン（Potts, $\mathbb{Z}_n$） | 有限群 Fourier $A\leftrightarrow\hat A$（Pontryagin） | $\mathbb{Z}_2$, Potts $e^{K_c}-1=\sqrt q$ |
| 一般 | Poincaré⊗Pontryagin、$H^k(X,A)\cong H_{d-k}(X,\hat A)$ | ホモロジー的 |
| 高次形式ゲージ（Wegner） | $(p,d)\mapsto(d-p-2,d)$ | $d=2p+2$ |
| 現代 | 有限（高次）対称性のゲージング・非可逆対称性 | 自己双対点の対称性 |

- KW＝「有限アーベル群の Fourier × セル複体の Poincaré 双対」の最小例。2D Ising が特別なのは **$d=2p+2$（$p=0\Rightarrow d=2$）の自己双対ケース**だから。
- 3D Ising（$p=0,d=3$）→ 3D $\mathbb{Z}_2$ ゲージ（$p=1$）。**別理論に写る**ので不動点論法が効かず、臨界点は代数的に固定されない。
- 一般論はすべて有限群・複体の組合せ＝**可算/代数的**。
- 非可換群では「双対群」が無く、双対は表現圏（fusion category）の言葉になる（アーベルが綺麗）。

## 4. 双対が買うもの／買わないもの

- **閉形式の存在**：✕。双対は 1 本の関数等式にすぎず $f$ を決定しない。Onsager の閉形式は転送行列由来で**独立**。
- **相転移の存在**：✕。双対は存在を証明しない（自己双対点が解析的でも両立）。
- **買うもの**：転移の**位置の条件付き固定**。$(\text{存在})\wedge(\text{一意})\Rightarrow$ 自己双対点（代数的 $\overline{\mathbb{Q}}$）。

役割分担（独立な三成果）：Peierls（**存在**, 1936）／KW 双対（**位置**, 1941）／Onsager・Yang（**自由エネルギー・指数**, 1944–52）。

**この枠組みとの関係**：双対の払い戻しは「臨界点を代数的一点 $\overline{\mathbb{Q}}$ に固定」（可算側、ただし存在・一意性を仮定）。自由エネルギー・臨界指数の $\mathbb{R}$ 移行（極限）は双対では消えず、[07](07_Λ帰属と可解性は別.md) の「(3) 可解性は別構造から来る」と整合 ── 双対は (3) の半分（位置）しか埋めない。
