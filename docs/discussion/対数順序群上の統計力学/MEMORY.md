## 確定済みの土台

- 土台は**対数順序群** $\Lambda=\bigoplus_p\mathbb{Z}\ell_p$（素数の対数が張るアルキメデス順序アーベル群）。$\log:\mathbb{Q}_{>0}\to\Lambda$ は素因数分解。順序は整数比較で判定可能。体 $L$ は不使用（割り算が要る所＝温度は $\beta$ の比較に還元）。
- $\Lambda$ を出る本物の脱出（指数評価・$N\to\infty$・微分）は中間体を経ず一気に $\mathbb{R}$。Fisher 零点の $\overline{\mathbb{Q}}$ は別建ての可算構造（$\mathbb{Z}[x]$ の分解体）。
- 旧称「指数体 $E$」は廃止。$\overline{\mathbb{Q}}^{\exp}$ は不要と判明。
