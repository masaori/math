# 07. 帰属・計算可能性・複雑性・可解性は別物

**問い**：有限 $N$ では $\Lambda$ に居られると分かったが、「極限を取れるところまで計算が進むか」は別問題ではないか。

**要点（結論先出し）**：その通り。しかも分けるべき軸は**四つ**ある。$\Lambda$ が保証するのは一番目（帰属）だけで、それすら無条件ではない。意味のある独立は「有限の複雑性」と「極限の閉形式」の間にある。

## 1. 四つの別物（言葉で）

扱う対象は二つ：**有限 $N$ の値**（$\Omega_N,Z_N$）と、**極限**（$-\beta f=\lim_N\frac1N\log Z_N$）。それぞれに別の問いが立つ。

1. **そもそも可算・整数の世界に住むか（帰属／存在）**
 — サイトが有限個・各サイトが離散（有限個の値）なら、配置数は正真正銘の整数で $\Omega_N\in\mathbb{N}$、$S_N=\log\Omega_N\in\Lambda$、$Z_N\in\mathbb{Z}[x]$、零点 $\in\overline{\mathbb{Q}}$。**だが無条件ではない**：スピンが連続（球面値・場の理論）、各サイトの状態が無限個、または無限サイズになると、最初から積分・無限和＝実数になり、可算の世界に住まない。**帰属は「有限かつ離散」のときだけ**○。
2. **有限時間で計算できるか（計算可能性）**
 — 有限・離散なら、全配置を数え上げれば必ず終わる ⟹ **つねに○**。イジングもスピングラスも全部○。**どれも同じ答えなので、模型の区別をしない自明な軸**。
3. **安く計算できるか（複雑性）**
 — 終わるとして手間が $N$ について多項式か指数か。平面イジングは多項式（Pfaffian）、3 次元・スピングラスは #P 困難。**模型ごとに変化する、有限側の本体軸**。
4. **極限に閉じた式があるか（可解性）**
 — オンサーガーのように在るか、3 次元のように無いか。**模型ごとに変化する、極限側の軸**。3 とは別。

$\Lambda$ 形式が言うのは **1（帰属）だけ**、しかも「有限・離散」のとき限定。2・3・4 については何も言わない。

## 2. 意味のある独立は「複雑性 × 閉形式」

「計算可能性と可解性が独立」と言うと不正確。**2（計算可能性）はつねに○で動かない**ので、4 と並べても情報がない（退化した独立）。意味があるのは **3（有限の複雑性）と 4（極限の閉形式）の独立**で、4 隅すべて実在する。

| | 極限に閉じた式が**ある** | 極限に閉じた式が**ない** |
|---|---|---|
| **有限が安い**（多項式） | **2 次元イジング**（平面→Pfaffian で多項式／Onsager の初等閉形式） | **1 次元ランダム鎖**（転送行列の積で多項式／極限＝ランダム行列積の Lyapunov 指数で一般に閉形式なし） |
| **有限が高い**（#P/NP 困難） | **SK スピングラス**（基底状態 NP 困難・分配関数 #P 困難／極限＝**Parisi 公式**で厳密に確定）、**REM**（指数個の準位／Derrida の初等閉形式）、**数の分割問題**（NP 困難／Mertens が極限を解いた） | **3 次元イジング**（#P 困難・Istrail 2000／閉形式は未解決の有名問題）、**3 次元スピングラス**（Edwards–Anderson／#P 困難・閉形式なし） |

**なぜ両立するか**：有限の難しさは「特定インスタンスを厳密に解く worst-case の手間」、極限は「典型・統計的挙動」── 別の対象。極限では自己平均化で個別インスタンスの難しさが**洗い流される**ことが多く、難しい有限でも極限が解けうる（SK→Parisi、REM）。逆に有限が自明に安くても極限に式がない（1 次元ランダム鎖）。

## 3. 「解けている」の粒度（初等 vs 変分）

「閉形式」には幅がある。
- **初等閉形式**：オンサーガー（積分一発）、REM（二領域の式）。
- **厳密だが変分形式**：Parisi 公式（レプリカ対称性の破れの汎関数最適化）、空洞法の方程式。**答えは確定しているが一行の初等式ではない**。

これを踏まえた正直な観察：
- **#P 困難 ＋ 厳密に解けた極限（変分形式）** は確実に在る（SK→Parisi）。
- **#P 困難 ＋ オンサーガー級の初等閉形式** という両取りの綺麗な例は**知られていない**。初等閉形式を生む構造（自由フェルミオン・平面性・対称性）は、たいてい**有限サイズも同時に楽にしてしまう**から。論理的には独立だが、現実には相関している。

## 4. 具体例の軸：Ising スピングラス

ランダム結合 $J_{ij}=\pm J$（quenched disorder）。
- **帰属**：各 disorder 実現で $\Omega_N\in\mathbb{N}$、$S_N\in\Lambda$。平均は $\Lambda_\mathbb{Q}$。有限・離散なので○。
- **計算可能性**：数え上げで終わる ⟹ ○（自明）。
- **複雑性**：3 次元または磁場ありで基底状態 NP 困難（Barahona 1982）、分配関数 #P 困難。（例外：2 次元・無磁場・平面は Pfaffian で多項式。）
- **可解性**：SK 平均場では Parisi 公式で極限が確定（変分形式）。3 次元短距離では未解決。

## 5. 含意

- **可積分・厳密可解模型 ＝ 4 が通る特別な模型**。$\Lambda$ 形式はそこに中立で、可解性は別構造（双対・Yang–Baxter・自由フェルミオン）から来る。
- $\mathbb{R}$ 移行（極限を取る）が「意味のある一点」になるには 4 が要る。**4 が無ければ移行先が定まらない**（数値的にしか近づけず、極限値が計算不能実数のこともある）。
- **正直さの利点**：可算枠は「データは厳密・可算（ただし有限・離散のとき）」と「解けるか」を明示的に分離し、後者を誇張しない。「$\Lambda$ で閉じる」は可解性の主張では**ない**。

一言：**帰属は「有限・離散」なら○（連続・無限だとバツ）。計算可能性は有限・離散なら全部○（区別しない）。区別するのは複雑性（多項式 vs #P 困難）と極限の閉形式で、この二つが独立。$\Lambda$ は器を保証するだけで、安く計算できることも解けることも保証しない。**
