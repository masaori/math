# 対数順序群上の統計力学

可算な**対数順序群 $\Lambda$** の上でできるだけ統計力学を定式化し、$\mathbb{R}/\mathbb{C}$ への移行（収束・連続極限）は不可欠な箇所に限定する、という立場での議論記録。

（旧称「指数体 $E$ 上の統計力学」。議論の結果、土台は指数体ではなく**素数の対数が張るアルキメデス順序アーベル群** $\Lambda$ で十分・適切と判明したため改称。経緯は [00](00_記号と定義.md) と [docs/discussion/可算性の効用/](../可算性の効用/README.md)。）

## 土台の集合

$\mathcal{P}=\{2,3,5,\dots\}$ を素数全体、各 $p$ に形式記号 $\ell_p$ を割り当てる。

$$
\Lambda:=\bigoplus_{p\in\mathcal{P}}\mathbb{Z}\,\ell_p
$$

— **対数順序群**（アルキメデス順序付き自由アーベル群、$\mathbb{Z}$ 係数）。$\log:\mathbb{Q}_{>0}\to\Lambda$ の像。エントロピー・逆温度・Massieu 自由エントロピーがここに住む。順序は **2 整数の比較**で判定可能（無条件・初等）。$\exp$ も完成 $\mathbb{R}$ も使わない。

## $\Lambda$ を出ると一気に $\mathbb{R}$（体 $L$ は経由しない）

$\Lambda$ から出る「本物の脱出」は指数評価・極限・微分で、いずれも**中間体を経ず一気に $\mathbb{R}$** へ飛ぶ。よって体 $L=\overline{\mathbb{Q}}(\ell_p)$ を中間に置く意味はなく、本議論では**使わない**。

- 温度 $T=1/\beta$ は形式上 $L$ に落ちるが、相等・大小は $\beta\in\Lambda$ の比較に還元するので作らない。
- 代数的エネルギー量子で割ると $\overline{\mathbb{Q}}$ 係数（$\bigoplus_p\overline{\mathbb{Q}}\ell_p$）に落ちるが、整数エネルギー（Ising 等）では起きない。
- Fisher 零点の $\overline{\mathbb{Q}}$ は「$\Lambda$ からの脱出」ではなく、整係数多項式 $\mathbb{Z}[x]$ の分解体（フガシティ平面）という**別の可算構造**。零点プログラム専用に並置する（[05](05_相転移はΦで論じられるか.md), [06](06_文献調査_Ising臨界指数.md)）。

## 立場（二層構造）

- $\Lambda$ のような**可算集合**でできるだけ議論し、不可欠な部分でだけ収束の議論をするため完備体（$\mathbb{R}/\mathbb{C}$）に移行する。
- したがって解析（微積分・収束・指数）は基本的に**正当化されていない**。それは具体値を計算するときの「外の世界」のツールと捉える。
- 核を一文で：**「数える・比を取る・$\log$ を正の有理数に当てる（＝素因数分解）」だけで第〇法則と $\Lambda$ 上カノニカルまで到達でき、解析は具体値を出すとき外の世界で呼び出すツールに退避させる。**

## 記述上の原則（この議論全体で厳守）

1. **数学的厳密性**：登場するすべての記号がどの集合に属するか、施される演算がどう定義されるかをすべて明示する。とくに $\Lambda$ と $\mathbb{R}/\mathbb{C}$ の分別（収束を議論できるのか否か）を常に併記する。$\Lambda$ ないし可算集合で議論し、$\mathbb{R}/\mathbb{C}$ への移行は不可欠な連続極限に限る。微積分を不用意に正当化しない。
2. **簡潔性とスコープ**：壁打ちのため素早いイテレーションを要する。質問・依頼された内容のみを過不足なく答える。

## $\Lambda$ を出るのが不可避な箇所（いずれも一気に $\mathbb{R}$）

1. **$\beta$ 定数化・指数丸め**（標準カノニカル）：指数評価 $\exp_\mathbb{R}(-\beta\varepsilon)\notin\Lambda$ で離脱。
2. **示量極限** $N\to\infty$。
3. **§6 の $r\ge2$ で量子なし**のときの方向微分。

（注：「$\beta$ 定数化」自体は $\Lambda$ 内の線形化近似で $\Lambda$ を出ない。境界は $\beta$ 定数化と指数評価の間。詳細は [01](01_β定数化はΛを出るか.md)。）
