# 可算性の効用

可算集合（とくに対数順序群 $\Lambda$、代数的数 $\overline{\mathbb{Q}}$、$\mathbb{Q}$、$\mathbb{N}$）で議論が完結することの**工学的・実務的ペイオフ**を扱う壁打ち。

## 位置づけ（三部作の第三ノート）

このリポジトリには既に関連する2つの研究ノートがある。本フォルダはその交点の第三軸。

- **基礎論「なぜ可能か」** → [docs/research/実無限を認めない数学基礎論.md](../../research/実無限を認めない数学基礎論.md)（逆数学・構成的・計算可能解析、$\mathbb{R}$ 過剰論）
- **「どの分野で効くか」** → [docs/research/有限記号操作で証明可能な分野.md](../../research/有限記号操作で証明可能な分野.md)（書換え・正規形・決定手続きの分野サーベイ）
- **「可算で完結する工学的効用」（本フォルダ）** → 決定可能性・厳密計算・形式検証・証明証明書という実利の整理

母体は [docs/discussion/対数順序群上の統計力学/](../対数順序群上の統計力学/README.md)（対数順序群 $\Lambda$ 上の統計力学）。そこで現れた $\Lambda/\mathbb{R}$ 境界を、統計力学に限らない一般原理として抽出する。

## 運用

- 壁打ちで議論を伸ばし、結論が固まったら [docs/research/](../../research/) の第三ノートへ昇格する。
- 記述原則は [CLAUDE.md](../../../CLAUDE.md) の「議論・記述の原則」に従う（集合帰属の明示、可算／$\mathbb{R}$ 分別、簡潔性）。
