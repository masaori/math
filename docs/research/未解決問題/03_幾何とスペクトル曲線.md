# 03 幾何・スペクトル曲線側の未解決問題

曲面群 $\pi_1(\Sigma_g)$ のスペクトル曲線（[`../曲面群スペクトル曲線_球面とトーラス/`](../曲面群スペクトル曲線_球面とトーラス/README.md)）まわり：Hitchin 系・指標多様体・可換多様体・Painlevé。状態タグ [E]/[部分]/[未解決]、arXiv 番号併記。

---

## 1. 高次元 Hitchin 写像（Chen–Ngô）[部分解決／完全可積分系は未解決]

**枠組み**（Chen–Ngô 2020, Duke 169, arXiv:1905.04741）：$\dim X\ge2$ で Hitchin 写像は素朴基底 $\mathcal{B}=\bigoplus_i H^0(X,L^{d_i})$ へ**全射しない**。可換性拘束で像は非線形な閉部分スキーム $\mathcal{A}_X\subsetneq\mathcal{B}$（**spectral base**）に落ち、$\mathcal{M}_{\mathrm{Higgs}}\to\mathcal{A}_X\hookrightarrow\mathcal{B}$ と分解。

**Chen–Ngô 全射性予想**：spectral data 射 $\mathcal{M}_{\mathrm{Higgs}}\to\mathcal{A}_X$ は全射（＝spectral 被覆の有限 Cohen–Macaulay 化の存在と同値）。

**証明済み**：
- $GL_n$・任意の代数曲面 — Song–Sun 2021/23（arXiv:2107.01679, IMRN）[E]
- rank 2・**任意次元**の射影多様体 — He–Liu 2023/24（arXiv:2310.18934, Proc. LMS、二重分岐被覆で CM 化）[E]
- 一般簡約 $G$・ruled 曲面/楕円 fibration — Huynh 2025（arXiv:2504.01874）[E]
- **K-trivial／r-small 高次元** — Patel–Weissmann 2026（arXiv:2604.03217、正規 spectral 被覆、強化版）[E]

**未解決**：一般 $G$・一般 $\dim X\ge3$ の全射性 [未解決]。より深く、**$\dim X\ge2$ で代数的完全可積分系（ACIS）が存在するか**は本質的に未解決——spectral base が非線形・低次元、spectral/cameral 被覆が非平坦・非 CM、generic ファイバーがアーベル多様体でなく、大域正則シンプレクティック Lagrangian fibration 構造が未知。**曲線の場合の綺麗な可積分系に相当するものは無い**（[`../曲面群スペクトル曲線_球面とトーラス/03 §4`](../曲面群スペクトル曲線_球面とトーラス/03_任意次元トーラス_Zd.md)）。

**RΛ 接続**：障害はまさに**可換スキームの不変式論・Weyl 偏極**＝下記 §2 の可換 $d$-組の可約性。$T^d$ ($d\ge3$) のアーベル多様体上 Higgs 場が可換 $d$-組であることと直結。

## 2. 可換 $d$-組多様体の可約性 [未解決ギャップ]

$C(k,n)=\{$可換 $k$-組の $n\times n$ 行列$\}$（$k$＝組長、$n$＝行列サイズ、char 0）。

| $k$ | 既約性 | 状態 |
|---|---|---|
| **2**（対）| 全 $n$ で既約（$\dim=n^2+n$）| [E] Motzkin–Taussky 1955, Gerstenhaber 1961 |
| **3**（三つ組）| $n\le10$ 既約（$n=11$ は announced 未公刊）/ $n\ge29$ 可約 | **開ギャップ $11\le n\le28$** |
| **$\ge4$** | $n\le3$ 既約 ⟺、$n\ge4$ 可約 | [E] Gerstenhaber 1961（ギャップ無し）|

**三つ組の閾値の系譜**（$n$ 昇順で既約）：Guralnick 1992（$n\le3$）→ Guralnick–Sethuraman 2000（$n=4$）→ Holbrook–Omladič 2001（$n=5$）→ Omladič 2004（$n=6$）→ Han 2005（$n=7$）→ Šivic 2008（$n=8$）→ Šivic 2012（$n=9,10$）。可約側：Guralnick 1992（$n\ge32$）→ Holbrook–Omladič 2001（$n\ge30$）→ **Ngô–Šivic 2014（$n\ge29$）**。

**Gerstenhaber 問題**（1961、$C(3,n)$ は既約か）は**開ギャップ $11\le n\le28$** に縮小。**Guralnick–Omladič 予想**（$n>7$ で可約）は**反証済み**（$n=8,9,10,11$ 既約）。現在「最小の可約 $n$ を求めよ」が未解決（Jelisiejew–Šivic 2022, Crelle 788, arXiv:2106.13137）。

**冪零版** $N(3,n)$：$n\le6$ 既約・$n\ge13$ 可約・**開ギャップ $7\le n\le12$**（Ngô–Šivic 2014, arXiv:1308.4438）。$N(2,n)$ は全 $n$ 既約（Baranovsky 2001）。

**現代の前線**：Jelisiejew–Šivic が $n\le7$ の全既約成分を分類、$k\ge4,n\ge8$ で**生成的に非被約な成分**を証明。問題は「$C(3,n)$ は可約か」から「非主成分と特異点の記述」へ。Hilbert/Quot スキーム（$\mathrm{Hilb}^n(\mathbb{A}^d)$）と密結合。

**RΛ 接続**：$d=2$ でトーラスが綺麗なのは $C(2,n)$ が全 $n$ で既約だから（[`../曲面群スペクトル曲線_球面とトーラス/03 §3`](../曲面群スペクトル曲線_球面とトーラス/03_任意次元トーラス_Zd.md)）。$d\ge3$ の可約性が高次元 Hitchin（§1）の非可積分性の代数的核。

## 3. P=W 予想 [GL_n 解決／一般 $G$ 未解決]

**主張**（de Cataldo–Hausel–Migliorini 2012, Annals 175, arXiv:1004.1420）：非可換 Hodge 同型の下、Dolbeault の Hitchin 写像の perverse filtration ＝ Betti 指標多様体の weight filtration：$P_kH^*(M_{\mathrm{Dol}})=W_{2k}H^*(M_B)$。rank 2 で証明。

**解決（$GL_n$、全 rank、$g\ge2$）**：2022 に独立2証明——
- Maulik–Shen（arXiv:2209.02568, Annals 200 (2024)）：tautological 類の perversity＋Mellit の curious hard Lefschetz。
- Hausel–Mellit–Minets–Schiffmann（arXiv:2209.05429）：$\mathcal{H}_2$ 作用経由（放物 rank 任意）。

**未解決**：
- **一般簡約 $G$**（$SO_n, Sp_{2n}, G_2$, 例外群）[未解決]——技法は type A（Springer 理論）特化。
- **PI=WI（特異・交叉コホモロジー）**[部分]：Felisetti–Mauri が $g=1$ 任意 $n$・$g=2,n=2$。一般未解決。
- **wild/irregular P=W**[未解決]（下記 §5）。

**関連解決**：curious hard Lefschetz（HLRV 予想）は **Mellit 2019（arXiv:1905.10685）が証明**（smooth generic、$\gcd(n,d)=1$、cell 分解）[E]。

## 4. 指標多様体のコホモロジー [部分／未解決]

- **HLRV mixed Hodge 予想**（Hausel–Letellier–Rodriguez-Villegas 2011, Duke 160, arXiv:0810.2076）：genus $g$・$k$ 穴（generic 半単純類）の完全 mixed Hodge 多項式 $H(M_B;q,t)$ の Macdonald 多項式による閉形式。**$E$-多項式（$t=-1$）は証明**（点数と一致）[E]、**完全 $(q,t)$ 版は未解決**。purity 予測も部分的。
- **$\mathbb{Z}^d$（可換多様体の指標スタック）**：Florentino–Silva（arXiv:1711.07909）が一般簡約 $G$ の $E$-多項式を明示 [E]。$d\ge3$ で表現多様体が可約・高特異ゆえ、**完全指標多様体の mixed Hodge・cohomological integrality は未解決**（$GL/SL$・小 $d$ 超）。
- **cohomological integrality / BPS**（Davison ら）：character stack の BPS 分解。強 cohomological integrality（arXiv:2509.21298, 2025）が smooth symmetric stack・3-manifold で部分進展 [部分]。一般 $G$・特異 $\pi_1(\Sigma_g)$ の完全 IH 版は未解決。

## 5. Painlevé・Garnier・wild・Deligne–Simpson

**PVI＝4穴球面（[`../曲面群スペクトル曲線_球面とトーラス/02 §4`](../曲面群スペクトル曲線_球面とトーラス/02_球面と穴あき球面_自由群.md)）**：
- 非線形モノドロミー＝modular 群作用（Fricke–Klein 立方面）[E] Iwasaki 2002/03, Cantat–Loray 2009。
- **代数解の完全分類**[E]：Lisovyy–Tykhyy 2014（JGP 85, arXiv:0809.4873）——4連続族＋**45例外解**（Dubrovin–Mazzocco 三角群、Boalch 52 icosahedral、Klein 解）。近年 computer 検証（arXiv:2407.21180、Tykhyy 予想 arXiv:2409.04379）。
- RH 写像＝解析的同型・幾何 Painlevé 性 [E] Inaba–Iwasaki–Saito 2006、van der Put–Saito 2009。Okamoto/Sakai 曲面 [E] Sakai 2001。

**未解決**：
- **正則 Garnier $N\ge2$ の代数解完全分類**[未解決]（irregular は Diarra–Loray 2020 arXiv:1808.09190 が largely 解決、pull-back/classical）。「非線形 Schwarz list for $G_N$」が未証明。
- **高階局所系の有限軌道**（Landesman–Litt 2024, arXiv:2409.02234）[未解決 rank $\ge3$]：有限 MCG 軌道は「幾何/pull-back 由来」であるべき。
- **wild mapping class 群の予想**（Boalch; Doucot–Rembado 2024–25）[部分／未解決]。
- **wild P=W**[未解決（一般）]：Painlevé 場合は Szabó（arXiv:1802.03798, 1906.01856）・Némethi–Szabó（arXiv:2001.11720）が幾何版証明 [部分]、全 irregular type/全 rank は未解決。
- **Deligne–Simpson**：additive/tame は Crawley-Boevey 2003（Duke 118、Kac–Moody 根）解決 [E]。**分岐（ramified）irregular は未解決**（unramified は Hiroe 2017 Duke 166 解決、maximally ramified は KLMNS 2022 arXiv:2108.11029 の特殊例のみ）。

## 6. 上位8（幾何側）

1. 高次元 Hitchin の完全可積分系存在（Chen–Ngô、$\dim\ge2$）[未解決]
2. P=W 一般簡約 $G$（$GL_n$ は 2022 解決）[未解決]
3. 可換三つ組 $C(3,n)$ の可約性ギャップ $11\le n\le28$ [未解決]
4. HLRV mixed Hodge 予想（$E$-多項式は証明）[未解決]
5. 正則 Garnier $N\ge2$・高階有限軌道（Landesman–Litt）[未解決]
6. wild P=W の一般化 [未解決]
7. 分岐 irregular Deligne–Simpson [未解決]
8. 特異指標多様体の cohomological integrality／PI=WI [部分]

**recently resolved（対照）**：P=W for $GL_n$（Maulik–Shen, HMMS 2022）、curious hard Lefschetz（Mellit 2019）、PVI 代数解分類（Lisovyy–Tykhyy 2014）、rank2 全次元 Hitchin 全射性（He–Liu 2023）。
