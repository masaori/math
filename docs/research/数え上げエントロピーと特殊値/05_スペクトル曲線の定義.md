# 05 スペクトル曲線の定義（周期的作用素の Bloch–Fourier 変換）

自由エネルギー $=m(P)$ の $P$、すなわちスペクトル曲線を定義から厳密に。全トークンの所属集合を明示。

## 1. 周期的作用素

2 次元格子模型を両方向に平行移動不変な形で置く。基本領域あたりの内部状態数 $d_0\in\mathbb{N}_{>0}$、状態空間 $\ell^2(\mathbb{Z}^2)\otimes\mathbb{C}^{d_0}$。模型の局所構造（転送行列／Kasteleyn 作用素／Laplacian）はこの空間上の**有限レンジ・$\mathbb{Z}^2$-周期的線型作用素** $A$ を定める。有限レンジ＝行列要素 $A_{(m,n),(m',n')}$ は $|m-m'|,|n-n'|$ 有界のときのみ非零。

## 2. Bloch–Fourier 変換

平行移動 $S_1,S_2$（水平・垂直シフト）は $A$ と可換 $[A,S_1]=[A,S_2]=0$。同時対角化＝Bloch–Fourier 変換：$S_1,S_2$ の同時固有値 $(z,w)\in(\mathbb{C}^\times)^2$（**Bloch 位相**、$|z|=|w|=1$ が物理的）で $A$ は各点で有限行列 $A(z,w)\in M_{d_0}(\mathbb{C})$ に落ちる。有限レンジゆえ各要素は Laurent 多項式：
$$A(z,w)_{ab}=\sum_{(k,l)\ \text{有限}}A^{(k,l)}_{ab}z^kw^l\in R[z^{\pm1},w^{\pm1}],$$
$R$＝係数環（整数重みで $\mathbb{Z}$）。$z^kw^l$＝基本領域を $(k,l)$ ずらす寄与＋Bloch 因子。

## 3. 定義：スペクトル多項式と曲線

$$\boxed{P(z,w):=\det A(z,w)\in R[z^{\pm1},w^{\pm1}]}\qquad\boxed{C:=\{(z,w)\in(\mathbb{C}^\times)^2:P(z,w)=0\}}$$
$C$＝$(\mathbb{C}^\times)^2$ 内平面代数曲線。**Newton 多角形** $\Delta\subset\mathbb{R}^2$＝$P$ の単項指数 $(k,l)$ の凸包＝組合せ的骨格。

## 4. 固有値・分配関数・自由エネルギー

固定水平 Bloch 位相 $z$（$|z|=1$）で垂直転送行列 $T(z)$ の固有値は $P(z,\cdot)$ の $w$-根：$P(z,w)=0\iff w=\lambda_i(z)$（$i=1,\dots,d_0$）。$L\times N$ トーラス：
$$Z_{N,L}=\sum_{z^L=1}\sum_{i=1}^{d_0}\lambda_i(z)^N=\sum_{z^L=1}\mathrm{Tr}(T(z)^N)\in R.$$

**自由エネルギー**：
- 行列式（dimer/Kasteleyn）模型では厳密に $-\beta f=\lim\frac1{NL}\log Z=m(P)=\int_{\mathbb{T}^2}\log|P|\,d\mu$（Kasteleyn＋KOS）。
- 一般の正転送行列では $-\beta f=\int_{|z|=1}\log|\lambda_{\max}(z)|\frac{|dz|}{2\pi}$（各 Bloch 位相の Perron 根）；$P$ が $w$-monic で支配根が単位円外に一つのとき $m(P)$ に一致。厳密な一般命題は「$P$ が定める $\mathbb{Z}^2$-代数力学系の位相的エントロピー $=m(P)$」（Lind–Schmidt–Ward）。

## 5. スペクトル曲線が符号化するもの

| $C$ の属性 | 統制 |
|---|---|
| Newton 多角形 $\Delta$ | 高さ関数の傾き・cluster 可積分系（Goncharov–Kenyon）|
| amoeba $\mathrm{Log}(C)$ | 相図（frozen/liquid/gas）|
| 種数（＝amoeba の穴の数）| ガス相数・特殊値からの隔たり |
| Harnack 極大性 | dimer スペクトル曲線であること |
| 算術（種数・CM・$L$ 函数）| 自由エネルギーが特殊値 $L'(E,0)$ か |
| $p$ 進 Newton 多角形 | $\mu_{\min}(p)$（R-Λ 双対 $\Lambda$ 側）|

## 6. 具体例

- **蜂の巣 dimer（種数0＝有理）**：二部・基本領域白黒 1 頂点ずつ・3 辺方向重み $a,b,c$。Kasteleyn は $1\times1$：$P(z,w)=a+bz+cw$。曲線は直線＝有理。$m(a+bz+cw)$＝Cassaigne–Maillot（dilog、対称で Catalan）。有限円筒 $W=4$ の $Z_N=2\cdot4^N+1$ とは別（後者は有限体積、スペクトル曲線は無限体積）。
- **Ising（種数1＝楕円）**：$P_K(z,w)=\cosh^2 2K-\frac12\sinh2K(z+z^{-1}+w+w^{-1})$。$zw$ 倍で双二次＝楕円、臨界で退化。$m(P_K)=-\beta f$（Onsager）、特殊温度で $L'(E,0)$（Viswanathan 2024）。
- **六頂点**：$T(z)\in M_{d_0}(\mathbb{Z}[z^\pm])$、$P(z,w)=\det(wI-T(z))$。

## 7. R-Λ 双対での位置

同一整数 $P$ を二素点で読む：$\infty$ で $m(P)$（自由エネルギー、アルキメデス）、各 $p$ で $P\bmod p$ 点数・$p$ 進 Newton 多角形（$\mu_{\min}$、$\Lambda$ 側）。Newton 多角形が両トロピカル影の共通骨格（箱玉系＝アルキメデス・トロピカル隅、`R-Lambda-duality/` §9.5）。
