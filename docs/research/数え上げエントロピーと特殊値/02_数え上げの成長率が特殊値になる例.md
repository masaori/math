# 02 数え上げの成長率が特殊値になる例

数え上げ・統計力学の成長率（エントロピー）が Mahler 測度・$L$ 値・周期になる具体例。[E]＝定理、[C/O]＝予想/未解決。

## 1. 全域木：tree entropy ＝ Laplacian の Mahler 測度

**一般定理 [E]**: 有限グラフ列が無限頂点推移グラフ $G$ に局所収束すると全域木数 $\tau$ が $\frac1{|V_n|}\log\tau\to h(G)$（tree entropy）＝Laplacian の正規化 (Fuglede–Kadison) 行列式の対数（Lyons 2005; Burton–Pemantle 1993）。$\mathbb{Z}^d$ で symbol $\Delta(z)=2d-\sum(z_j+z_j^{-1})$、
$$h_d=m(\Delta)=\int_{[0,1]^d}\log\Big(2d-2\sum\cos2\pi\theta_j\Big)d\theta.$$

**$\mathbb{Z}^2$ の値 [E]（Dirichlet $L$ 値）**:
$$h_2=\frac4\pi G=\frac4\pi\beta(2)\approx1.16624,\quad G=\beta(2)=L(\chi_{-4},2)\ (\text{Catalan}).$$
（Wu 1977; Shrock–Wu 2000; Lyons の行列式枠内。）

**高次元 [C/O]**: $h_3\approx1.6733$, $h_4\approx2.0000$（周期だが標準 $L$/$\zeta$ 値の閉形式は未証明）。$h_4=2$ ちょうどは**恐らく偽**（Rosengren ほか）。三角・蜂の巣格子の tree entropy は $\frac1\pi\times$(dilog/Clausen) で $L$ 値に還元（Shrock–Wu, Chang–Shrock）。

## 2. dimer/完全マッチング：自由エネルギー ＝ Kasteleyn スペクトル曲線の Mahler 測度

**一般定理 [E]**: 周期二部平面グラフの dimer 自由エネルギー $=-m(P)$、$P=\det K(z,w)$（Kasteleyn 作用素の特性多項式）。スペクトル曲線は常に **Harnack（極大実）曲線**（Kenyon–Okounkov–Sheffield 2006; 行列式漸近 Kenyon 2000）。

**正方格子 [E]（Catalan）**: $f_\square=\frac{G}{\pi}\approx0.29156$（Kasteleyn 1961; Temperley–Fisher 1961）。dimer あたり $e^{2G/\pi}\approx1.7916$。ドミノタイリング・Aztec diamond も同じバルクエントロピー。

**蜂の巣（菱形タイリング）・三角 [E]**: 蜂の巣 dimer ＝lozenge タイリング ＝boxed plane partition（MacMahon）。等方点はスペクトル曲線が退化（liquid/gas 境界）ゆえ Clausen/dilog 極限値。三角 dimer 自由エネルギーも $\frac1\pi\times$dilog（Fisher; Wu review 2006）。すべて周期、最も綺麗（正方）が $L$ 値。

## 3. 代数力学：位相的エントロピー ＝ Mahler 測度 ＝ 高さ

**Lind–Schmidt–Ward (1990) [E]**: $\mathbb{Z}^d$ 作用（コンパクトアーベル群自己同型、$f\in\mathbb{Z}[z^\pm]$）の位相的・測度論的エントロピー $=m(f)$。$d=1$ で $\sum_{|\alpha|>1}\log|\alpha|=m(f)$。

**Everest–Ward (1999) [E, 解説]**: 「エントロピー＝高さ」辞書（軌道成長 ↔ 算術的高さ、橋 $m(f)$）。全域木・dimer 行列式と力学エントロピーが同じ正規化行列式/高さである概念的理由。

**Deninger (1997)**: これらを正則子・Beilinson 予想へ持ち上げ（「なぜ $L$ 値か」の最深の説明、一部条件付）。

**Lehmer 問題 [C/O]**: 1 変数整数多項式の $\inf\{m(f)>0\}$ にギャップがあるか（$\ge\log1.17628$）＝最小正エントロピーのトーラス自己同型。未解決。

## 4. Ising：自由エネルギー ＝ Mahler 測度・$L$ 値

**Onsager (1944) [E]**: 正方格子零場自由エネルギー $=\log2+\frac1{8\pi^2}\iint\log[\cosh^2 2K-\sinh2K(\cos\theta_1+\cos\theta_2)]$＝スペクトル多項式 $P_K(z,w)=\cosh^2 2K-\frac12\sinh2K(z{+}z^{-1}{+}w{+}w^{-1})$ の Mahler 測度。$\{P_K=0\}$ は一般に楕円、臨界 $\sinh2K_c=1$ で退化（有理）。

**Viswanathan (2024, arXiv:2407.19531) [E]**: 正方・三角・蜂の巣の分配関数を楕円曲線 Mahler 測度に書き直し、超幾何閉形式を導出、**特殊温度で Hasse–Weil $L$ 値**・**臨界点で Dirichlet $L$ 値**を示す。Boyd–Deninger の Ising 実現。

**Boyd 多項式 [E]**: $1+x+\frac1x+y+\frac1y$（Ising/dimer 転送構造に密接）の Mahler 測度 ＝ 導手15 の $L'(E,0)$（Rogers–Zudilin 2014 で証明）。

**Ising ↔ dimer [E]**: 平面 Ising は装飾格子上の dimer（Fisher 1966, Kasteleyn）に写る。両自由エネルギーが同じ Kasteleyn 行列式＝Catalan/$L$ 値。

## 5. 他の数え上げ：代数的 vs 周期 vs 特殊値なし

| 問題 | 値 | 型 | 状態 |
|---|---|---|---|
| $\mathbb{Z}^2$ 全域木 | $4G/\pi$ | Dirichlet $L$ | [E] |
| 正方 dimer | $G/\pi$/site | Dirichlet $L$ | [E] |
| Ising 臨界 | Dirichlet $L$ | $L$ 値 | [E] (2024) |
| Ising 一般 $T$ | $m(\text{楕円})=rL'(E,0)$ | 楕円 $L$/周期 | [E]/Boyd 型 |
| 蜂の巣 SAW 連結定数 | $\sqrt{2+\sqrt2}$ | **代数的**（次数4）| [E] (D-C–Smirnov 2012) |
| hard hexagon | $\kappa\approx1.395486$ | **代数的**（次数24）| [E] (Baxter 1980) |
| square ice/$\mathbb{Z}^2$ 3彩色 | $(4/3)^{3/2}$ | **代数的** | [E] (Lieb 1967) |
| hard-square | $\approx1.503048$ | 閉形式なし | [C/O] |
| $\mathbb{Z}^2$ SAW 連結定数 | $\mu\approx2.638$ | 非代数的と予想 | [C/O] |
| 格子動物/polyomino（Klarner）| $\approx4.0626$ | "generic" | [C/O] |

**SAW**: 蜂の巣で連結定数が厳密に代数的 $\sqrt{2+\sqrt2}$（Nienhuis 1982 予想, Duminil-Copin–Smirnov 2012 証明, パラフェルミオン観測量）。$\mathbb{Z}^2$ では未知・generic 超越と予想。

**hard particles**: hard hexagon（三角格子隣接排他）は Baxter (1980) が厳密解、最大密度エントロピー定数は次数24の代数的数。hard-square は未解決・特殊値なしと目される。**代数性は Yang–Baxter 可積分＋有限種数曲線の指紋**（dimer/tree の超越 $L$ 値と対照）。

**彩色/Potts**: 零温反強磁性 Potts＝彩色多項式、per-site 成長 $W(q)$ は可解格子で代数的（square ice の $(4/3)^{3/2}$ が典型, Lieb 1967）。

**格子動物**: Klarner 定数は存在するが閉形式予想なし。「ただの実数」成長定数の原型。

## 6. 統一像と限界

$\text{エントロピー}=\text{正規化行列式}=m(P)=\text{高さ}$ が繰り返し現れ（LSW/Lyons/KOS/Everest–Ward/Deninger）、$m(P)$ は常に周期。どの細かいクラスかは $\{P=0\}$ の幾何:
- **種数0** → Dirichlet $L$（Catalan 等）、Smyth template。
- **種数1** → $rL'(E,0)$（Boyd）、Rogers–Zudilin で証明、Viswanathan で物理実現。
- **高種数/高次元** → 一般周期・（予想的）motive の $L$ 値。

**特殊値でないもの**: 可積分だが非自由フェルミオン（hard hexagon, square ice, 蜂の巣 SAW）は**代数的**；非可積分（hard-square, $\mathbb{Z}^2$ SAW, 格子動物）は**特殊値構造なし**。

**分類研究の現状**: どの成長定数が特殊値かを分類する定理はない。あるのは Boyd–Rodriguez-Villegas プログラム（modular 函数でパラメトライズした族→Eisenstein–Kronecker→CM/楕円 $L$ 値、大半予想＋多数の証明例）、Deninger の motivic/Beilinson 枠（条件付）、Chinburg 型予想（全 Dirichlet $L'$ が Mahler 測度）。
