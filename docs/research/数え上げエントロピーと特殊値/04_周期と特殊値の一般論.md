# 04 周期と特殊値の一般論

格子統計力学・格子和・Feynman 積分の数え上げ定数が**周期・$L$ 値・特殊値**になる現象の一般論と分類プログラム。[E]/[C/O]。

## 1. Kontsevich–Zagier 周期の枠組み

**定義 [E]**（Kontsevich–Zagier 2001）: **周期**＝実部・虚部が $\mathbb{Q}$ 係数有理関数の半代数的領域（$\mathbb{Q}$ 係数多項式不等式）上の絶対収束積分の値。同値に、$\overline{\mathbb Q}$ 上の多様体の代数的 de Rham 類と Betti 類のペアリング。

**環構造 [E]**: 周期の集合 $\mathcal P$ は可算な $\overline{\mathbb Q}$-代数で $\overline{\mathbb Q}\subsetneq\mathcal P\subsetneq\mathbb C$（$\pi,\log2,\zeta(3)$ を含むが可算＝ほとんどの複素数は周期でない）。拡大: $\widehat{\mathcal P}=\mathcal P[1/\pi]$、指数周期 $\mathcal P^e$（$e,\gamma,\Gamma(p/q)$）。$\gamma$ は周期と**知られていない**。

**例 [E]**: $\pi,\log\alpha,\zeta(n)$、**多重ゼータ値（MZV）**（$\mathbb P^1\setminus\{0,1,\infty\}$ の反復積分）、Dirichlet $L$ 特殊値、**完全楕円積分** $K(k),E(k)$（楕円曲線の周期）、**Gamma 値** $\Gamma(p/q)^q$（CM アーベル多様体）。

**KZ 周期予想 [C/O]**: 同一周期の二つの積分表示は加法性・変数変換・Stokes の有限鎖で結ばれる（「予期せぬ関係はない」）。motivic 版が Grothendieck 周期予想（§5）。Nori/Ayoub が**函数的**類似を証明。

**位置づけ**: 格子統計力学・格子和・Feynman 図の閉形式定数は驚くほど規則的に周期（拡大/指数周期）。研究は**どの周期か**（Tate/MZV vs 楕円 vs modular vs CY；weight；Galois 軌道）を motivic filtration で特定すること。

## 2. 格子和・格子 Green 函数

**格子和 [E]**: Glasser–Zucker corpus; Borwein–Glasser–McPhedran–Wan–Zucker *Lattice Sums Then and Now* (2013)。**Epstein zeta/Chowla–Selberg [E]**: 判別式 $-d$ の二次形式の 2 次元 Epstein zeta は Dirichlet $L$ の積に分解、特殊値は Chowla–Selberg で $\Gamma(a/d)$ 積＝**CM アーベル多様体の周期**。2 次元 Madelung 型は $\beta(2)$（Catalan）等の $L$ 値、3 次元 Madelung 定数は閉形式未知。

**格子 Green 函数（Watson）[E]**: Watson (1939) が sc/bcc/fcc の band edge を閉形式化: $W_{sc}=\frac{\sqrt6}{32\pi^3}\Gamma(\frac1{24})\Gamma(\frac5{24})\Gamma(\frac7{24})\Gamma(\frac{11}{24})$、bcc $=\Gamma(1/4)^4/(4\pi^3)$——**Gamma 値（CM）周期**。次元横断（Guttmann 2010）: 2D＝$_2F_1$/楕円積分（楕円周期）、3D＝$K^2$（band edge は Gamma 周期）、4D＝**Calabi–Yau** クラス、高次元＝高階 Fuchsian ODE。

**Bessel moment [E]**: $\int_0^\infty t^k I_0^a K_0^b\,dt$ が 2D 格子/ランダムウォーク定数かつ位置空間 Feynman（sunrise/banana）積分。Bailey–Borwein–Broadhurst–Glasser (2008): $\le6$ 個の Bessel の moment を $\Gamma$ 値・楕円積分・（高次で）modular $L$ 値で閉形式化。Bailey–Borwein–Crandall (2006): Ising class 積分 $C_n$、$C_\infty=2e^{-2\gamma}$（指数周期）。

## 3. Feynman 積分＝周期

- **Broadhurst–Kreimer (1995–97) [E]**: $\phi^4$ の 7 ループまでの繰り込み定数が MZV の $\mathbb{Q}$ 線形結合、既約 MZV 数が正結び目の数え上げと一致。cyclotomic 一般化（6 乗根）も。
- **Bloch–Esnault–Kreimer (2006) [E]**: Feynman 振幅＝グラフ超曲面補集合の（相対）コホモロジーの周期（motivic 定義）。Belkale–Brosnan: グラフ超曲面は motivic に最一般——**全 Feynman 周期が MZV とは限らない**（Brown–Schnetz が 8 ループで非 MZV=modular 例）。
- **Brown (2012) [E]**: $\mathbb{P}^1\setminus\{0,1,\infty\}$ の mixed Tate motivic $\pi_1$ が MZV で生成＝「**$\mathbb Z$ 上 mixed Tate motive の周期は MZV**」。グラフの周期が MZV ⟺ motive が mixed Tate unramified。
- **楕円・modular [E]**: sunset＝elliptic dilog（Bloch–Vanhove 2015）、higher normal function（Bloch–Kerr–Vanhove 2015）、Bessel moment Feynman 積分＝modular form の critical $L$ 値（Zhou 2018）、Bessel moment 間の二次関係の motivic 証明＝Kloosterman connection の対称冪の周期ペアリング（Fresán–Sabbah–Yu 2023）。

**Feynman が niceness の模型**: 周期が「nice」＝ motive が小さい/特殊（mixed Tate ⇒ MZV、modular form ⇒ modular $L$、CY）。coaction/cosmic Galois がどの周期が現れうるか制約（§5）。

## 4. RG/臨界振幅：統計力学定数は周期か

**自由エネルギー・臨界振幅 [E]**: Onsager 臨界自由エネルギー、dimer エントロピー（Kasteleyn: $G/\pi$, Catalan）等は Catalan・$\zeta(3)/\pi$＝周期/$L$ 値。自発磁化は代数的。solved-model 側の臨界振幅・自由エネルギー定数は確立された周期（自由フェルミオン可解ゆえ低複雑度）。

**Ising 感受率 [E/C/O]**: $\chi=\sum_n\chi^{(n)}$、各 $n$-粒子寄与は $n$ 重積分。Maillard グループ: $\chi^{(3)},\chi^{(4)}$ の Fuchsian ODE（Zenine–Boukraa–Hassani–Maillard 2004–05）、experimental math（Boukraa ほか 2008）、**globally nilpotent**（arXiv:0812.4931）、diagonal 感受率＝楕円積分・modular form・CY 方程式（arXiv:1110.1705）。**Bostan–Boukraa–Christol–Hassani–Maillard (2013) [E]**: $\chi^{(n)}$ は**有理函数の diagonal**（$G$-函数/周期、整数級数係数）。「これら統計力学量は周期」の最鋭の一般定理。歴史的 anchor: Wu–McCoy–Tracy–Barouch (1976) の Painlevé III。

**開フロンティア [C/O]**: 完全な $\chi$ は Nickel の**自然境界**（$\chi^{(n)}$ 特異点の稠密集積）をもち、格子上で **holonomic でない**と予想（各 $\chi^{(n)}$ は holonomic・周期だが総和はどの単一 motive の周期でもないかも）。「各次数は周期、総和は周期でないかも」の生きた例。

## 5. どの定数が特殊値かを分類するプログラム

**motivic/周期の Galois 理論 [C/O 核・[E] 実例]**: **Grothendieck 周期予想**——motive $M$ の周期体の超越次数 ＝ motivic Galois 群 $G_M$ の次元（全関係が motivic 起源）。周期に Galois 作用（André, Kontsevich, Nori, Ayoub; Huber–Müller-Stach *Periods and Nori Motives* 2017）。Nori/Ayoub が函数的類似を証明、算術版は開。

**cosmic Galois/coaction（物理の分類エンジン）[C/O]**: Brown, *Feynman amplitudes, coaction principle, cosmic Galois group*（arXiv:1512.06409）。各振幅に motivic 周期族、motivic coaction $\Delta$（Galois coaction）が張る空間に作用し、**coaction 原理**でこれが振幅上閉じる＝Cartier の cosmic Galois 群。$\phi^4$ で実証（Brown–Schnetz 2016）、各ループ次数でどの weight/MZV が現れうるか**計算前に予言**。既存の「どの数え上げ定数が特殊値か」の最も成功した分類。

**$G$-函数/diagonal/globally nilpotent（統計力学エンジン）[E/C/O]**: Maillard プログラム＝微分 Galois 型による分類。格子量の Fuchsian ODE を同定・因数分解し motive（有理/代数・$K$/楕円・modular・CY）に対応。diagonal-of-rational-functions 定理（2013）、globally nilpotent 分類（2008）、Christol 予想（globally bounded $G$-函数 ⟺ 有理函数の diagonal）。**格子量が nice 特殊値/周期 ⟺ holonomic ODE が算術的（globally nilpotent）で小 motive**——Feynman と同型の像。

**実験数学の feeder [E as method]**: 高精度数値＋整数関係検出（PSLQ）で閉形式候補を発見、motivic 理論が説明・証明。Broadhurst の Kloosterman moment $L$ 値予想 → Fresán–Sabbah–Yu/Zhou で証明が典型的 discovery→proof。

## 6. niceness の梯子（統合）

| 複雑度 | 定数 | 現れる場 | 状態 |
|---|---|---|---|
| 代数的/$\pi,\log$ | 磁化・単純振幅 | 全分野 | [E] 周期 |
| $\zeta(n)$, MZV, cyclotomic | 無質量 $\phi^4$ ~7 ループ | Feynman | [E] $\mathbb Z$ 上 MT motive（Brown）|
| $\Gamma(p/q)$（CM）| Watson bcc/sc, Epstein/Madelung 2D | 格子和/LGF | [E] Chowla–Selberg/CM |
| 楕円積分/$L(f,s)$ w2–3 | 2D/3D LGF, sunset, diagonal Ising $\chi$ | 全分野 | [E]（一部旧 [C/O]）|
| modular $L$ 値 w3–6 | Bessel moment Feynman, Ising blocks | Feynman+Ising | 大半 [E]（Zhou, FSY; Broadhurst [C/O] から）|
| Calabi–Yau 周期 | 4D+ LGF, 高次 $\chi^{(n)}$, banana | LGF+Ising+Feynman | [E] 同定、算術一部 [C/O] |
| 周期を超える? | **完全 Ising 感受率** $\chi$ | Ising | [C/O] 自然境界・非 holonomic 予想 |

**一文**: 格子和・LGF・可解 2D 模型の $n$ 重積分・収束 Feynman 積分の定数と holonomic 函数が**周期**であることは確立、相当数が motivic 型（Tate/MZV, CM-Gamma, 楕円, modular $L$, CY）に固定済み。motivic 型が「どの特殊値が現れうるか」を**分類**することは予想（Grothendieck/KZ・cosmic Galois/coaction・globally nilpotent $G$-函数が強く支持）。完全に総和した対象（完全 Ising 感受率）が周期クラスを逃れるかは開。
