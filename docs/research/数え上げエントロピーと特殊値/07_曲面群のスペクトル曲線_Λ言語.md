# 07 曲面群 $\pi_1(\Sigma_g)$ のスペクトル曲線（全部 Λ の言葉で）

種数 $g$ 曲面 $\Sigma_g$ の基本群 $\Gamma_g=\pi_1(\Sigma_g)$ に「対応するスペクトル曲線」を、$\mathbb{C}$ の Hitchin 幾何でなく**可算・決定可能な整数論（表現数・有限体点数）**で書く。Λ-native が本体、$\mathbb{C}$ の Hitchin スペクトル曲線はアルキメデスの顔。

**本ファイルは双曲 $g\ge2$ を扱う。両端 $g=0$（球面・穴あき）と $g=1$（トーラス $\pi_1=\mathbb{Z}^2$）の具体的スペクトル曲線は [`../曲面群スペクトル曲線_球面とトーラス/`](../曲面群スペクトル曲線_球面とトーラス/README.md) に別立て**（トーラス階数1＝Bloch トーラス上の $P(z,w)$＝RΛ 基底、階数 $N\ge2$＝楕円 Calogero–Moser；穴あき球面＝自由群 $F_{n-1}$、4 点穴＝Painlevé VI 楕円曲線）。

## 0. 基本群

$$\Gamma_g=\pi_1(\Sigma_g)=\langle a_1,b_1,\dots,a_g,b_g\mid\textstyle\prod_{i=1}^g[a_i,b_i]=1\rangle,$$
非可換・双曲（Fuchsian, 語の問題可解, 整数 $L^2$-Betti）。以下 $g=2$ を主に。

## 1. Λ-native な対象＝表現数（Frobenius–Mednykh）

有限群 $G$ への表現数：
$$\#\mathrm{Hom}(\Gamma_g,G)=\#\Big\{(a_i,b_i)\in G^{2g}:\prod[a_i,b_i]=1\Big\}=|G|^{2g-1}\sum_{\chi\in\mathrm{Irr}(G)}\chi(1)^{2-2g}\in\mathbb{Z}.$$
（Frobenius 1896, Mednykh。$g=2$：$|G|^{3}\sum_\chi\chi(1)^{-2}$。$\sum_\chi\chi(1)^{-2}=\zeta_G^{\mathrm{Witten}}(2)$＝Witten ゼータ。）

**$G$ の指標表という有限データだけ**から整数が出る——完全に可算・決定可能。$\Phi=\log\#\mathrm{Hom}(\Gamma_g,G)\in\Lambda$、素因数分解も決定可能。これが「$\Gamma_g$ の分配関数」の Λ-native な本体。

## 2. $G=\mathrm{GL}_N(\mathbb{F}_q)$：指標多様体の点数

$\#\mathrm{Hom}(\Gamma_g,\mathrm{GL}_N(\mathbb{F}_q))$ は $q$ の整数係数多項式＝**指標多様体 $X_{\Gamma_g,N}(\mathbb{F}_q)$ の点数**（Hausel–Rodriguez-Villegas, *Mixed Hodge polynomials of character varieties*, Invent. 2008）。これを $\bmod\ p$・$q=p^k$ で読むのが**Λ 側**；zeta 函数 $\prod_p(\dots)^{-1}$ の Euler 因子。点数多項式は palindromic（curious hard Lefschetz）で Macdonald 多項式に接続。

## 3. 整数スキームとしての指標多様体

$$X_{\Gamma_g,N}/\mathbb{Z}=\mathrm{Spec}\big(\text{trace 代数}\big)\big/\big(\textstyle\prod[a_i,b_i]=1\big),$$
trace 座標 $\mathrm{tr}(\rho(w))$（$w$＝語）の整数関係で定義（Procesi）。整数多項式、可算。**$\mathbb{Z}^d$ 版の「スペクトル多項式 $P\in\mathbb{Z}[z^\pm]$」の非可換対応が「指標多様体 $/\mathbb{Z}$」**。

## 4. Hitchin スペクトル曲線＝$\mathbb{C}$ の顔

非可換 Hodge（Hitchin–Simpson–Corlette–Donaldson）で
$$\mathrm{Hom}(\Gamma_g,\mathrm{GL}_N\mathbb{C})/\!\!/\mathrm{GL}_N\cong\mathcal M_{\mathrm{Higgs}}(\Sigma_g,\mathrm{GL}_N).$$
Higgs 束 $(E,\Phi)$, $\Phi\in H^0(\Sigma_g,\mathrm{End}E\otimes K)$ の**Hitchin スペクトル曲線**
$$\widetilde\Sigma=\{\det(\lambda-\Phi)=0\}\subset T^*\Sigma_g,\quad\lambda^N+a_1\lambda^{N-1}+\dots+a_N=0,\ a_i\in H^0(K^i),$$
$\Sigma_g$ の $N$ 重分岐被覆、種数 $N^2(g-1)+1$。Hitchin 写像 $\mathcal M_{\mathrm{Higgs}}\to B=\oplus H^0(K^i)$、ファイバー＝$\widetilde\Sigma$ の Jacobian/Prym（BNR 対応）、代数的完全可積分系（Hitchin 1987）。
- $g=2,N=2,\mathrm{SL}_2$：$\lambda^2=-a_2$（$a_2\in H^0(K^2),\dim 3$）、$\Sigma_2$ の二重被覆・分岐 4 点・$g(\widetilde\Sigma)=5$、Prym 次元 3。
- **$a_i$ を数体上に取れば $\widetilde\Sigma$ は $\mathbb{Z}$（or $\bar{\mathbb{Q}}$）上の曲線**、その点数・$L$ 函数が Λ 側。$\mathbb{C}$ の Higgs 束・hyperkähler・Hitchin ファイブレーションは**アルキメデスの顔**。

## 5. 唯一の $\mathbb{R}$ 脱出＝Witten 体積

同じ和 $\sum_\chi\chi(1)^{2-2g}$ を二素点で読む：
$$\underbrace{\text{有限 }G}_{\Lambda:\ \#\mathrm{Hom}\in\mathbb{Z},\ \text{指標表, 決定可能}}\ \longleftrightarrow\ \underbrace{G\text{ 連続化／}q\to\infty}_{\mathbb{R}:\ \text{平坦接続モジュライの symplectic 体積 (Witten 1991)}}.$$
Witten の 2D Yang–Mills 分配関数がこの和で、§06 の「双曲バンド理論＝2D YM（PRL 133, 146601, 2024）」に接続。R-Λ 双対のパターンそのもの。

## 6. 翻訳表

| $\mathbb{Z}^d$（可換）| $\Gamma_g=\pi_1(\Sigma_g)$（非可換）|
|---|---|
| 指標トーラス $(\mathbb{C}^\times)^d$ | 指標多様体 $\mathrm{Hom}(\Gamma_g,G)/\!\!/G$ |
| スペクトル多項式 $P\in\mathbb{Z}[z^\pm]$ | 指標多様体スキーム $/\mathbb{Z}$；Hitchin スペクトル曲線 $\widetilde\Sigma$ |
| 有限系＝1 の冪根特殊化 → $Z\in\mathbb{Z}$ | 有限群 $G$ 表現数 → $\#\mathrm{Hom}\in\mathbb{Z}$（Frobenius–Mednykh）|
| $\Phi=\log Z\in\Lambda$ | $\Phi=\log\#\mathrm{Hom}\in\Lambda$ |
| $\mu_{\min}$（$p$ 進 Newton）| $X(\mathbb{F}_p)$ 点数・zeta Euler 因子 |
| $m(P)$（$\infty$, Mahler 測度）| Witten 体積（$\infty$, $\sum_\chi\chi(1)^{2-2g}$ 連続版）|

## 7. 結論

**「$\pi_1(\Sigma_g)$ のスペクトル曲線」の Λ-native な姿は、有限群・有限体への表現数**（Frobenius–Mednykh／指標多様体の点数）＝指標表という有限データから決まる可算・決定可能な整数。$\mathbb{C}$ の Hitchin スペクトル曲線はそのアルキメデスの顔、Witten 体積が唯一の $\mathbb{R}$ 脱出。$\mathbb{Z}^d$ の「$P$ を 1 の冪根で特殊化→$Z\in\mathbb{Z}$→$\Phi\in\Lambda$、$\infty$ で Mahler 測度」が、$\Gamma_g$ で「指標多様体を有限群/有限体で数える→$\#\mathrm{Hom}\in\mathbb{Z}$→$\Phi\in\Lambda$、$\infty$ で Witten 体積」に平行移動する。

**注意（§06 と整合）**: ここで Λ 側が可算・決定可能なのは、曲面群が良い（語の問題可解・整数 $L^2$-Betti）から。失われるのは $\mu_{\min}$・$\mu_p$ の**有限素点法則**（$\mathbb{Z}[\Gamma]$ 非可換）であって可算性ではない。指標多様体の点数・zeta という別の可算構造がその代替候補。
