# Paper plan 001: 可積分だが極限未解決な格子模型の有限 N 可算決定可能性

## テーゼ（1文）

**Yang–Baxter 可積分だが熱力学極限が未解決の格子模型において、有限 N のすべての量（転送行列スペクトル・分配関数 / Massieu 自由エントロピー Φ・分配関数零点）は可算・決定可能な梯子 Λ/ℚ̄ に住み、明示的 witness（最小多項式・素因数分解）をもち、ℝ への移行は $N\to\infty$ の一点に隔離される。**

## 何が新しいか（正直な位置づけ）

- 個別の「有限行列の固有値は代数的」は初等。本稿の寄与は **(a) 可算/ℝ 梯子＋四軸での体系的 calibration**、**(b) 形式検証可能性（Λ＝整数比較／ℚ̄＝根分離, RCA₀, witness）**、**(c) これが非自明になる母集団の同定**＝「極限が未解決の可積分模型」（既解 Ising では自明）。
- すなわち **新しい格子模型の厳密解ではなく、可積分模型の有限サイズ理論の基礎論的・構造的再定式化**。可解性（極限の閉形式, 07-4）には中立で、それを誇張しない。

## 確立済み（SageMath 10.6 で検証済み, 本リポジトリ）

- XXZ 鎖（六頂点の対数微分）: 有限 N 固有値 ∈ QQbar・最小多項式 witness（`sagemath/check/C-U3_bethe_qqbar`）。
- 六頂点転送行列: $Z_{N,L}=\mathrm{Tr}\,T^N\in\mathbb{Z}$, $\Phi_N=\log Z_N\in\Lambda$, 非自明な数論構造（$v_2(Z_N)=N+2$ 等, D-U2）（`sagemath/check/D_phi_lambda`）。
- スピン1 Babujian–Takhtajan（高スピン可積分, 極限は Haldane で非自明）: 有限 N 固有値 ∈ QQbar, N=4 で次数2出現（`sagemath/check/apply_higher_spin_qqbar`）。

## 構成案

1. **序**: 可算/ℝ 梯子（ℕ⊂ℚ⊂Λ⊂ℚ̄ ⊂ ℝ）と四軸（帰属/計算可能性/複雑性/可解性, 07）。可積分=Bethe/スペクトル方程式が代数的。
2. **分離定理**: 有限・離散・可積分 ⇒ Ω∈ℕ, Z∈ℤ[x], Φ∈Λ, スペクトル/零点∈ℚ̄（決定可能・witness）。ℝ脱出は (a)(b)(c) のみ（00, 09）。
3. **未解決母集団での非自明性**: カイラル Potts / 高ランク・高スピン（カタログ `outputs/maps/integrable_unsolved_catalog.md`, McCoy/Baxter）。極限未解決でも有限 N は完全決定可能。
4. **worked examples**: XXZ・六頂点・スピン1 BT（上記検証）。Φ の数論構造（D-U2）。
5. **形式検証**: Λ=整数比較／ℚ̄=根分離 ⇒ decide/witness, RCA₀。Lean ターゲット仕様（`cycle1_05_formal_verification_spec.md`）。
6. **スコープと限界**: 可解性（極限閉形式）は別問題で本稿は主張しない。#P 側（3D/スピングラス）は有限でも計算困難で射程外（E）。
