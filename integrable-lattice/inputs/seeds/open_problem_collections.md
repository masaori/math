# Open Problem Collections

明示的に open problem が列挙されている、もしくは未解決状況の密度が高いソース。
harvest はこれらを最優先で拾い、extract は問題リスト形式から抽出する。

## Books / monographs

- Baxter, "Exactly Solved Models in Statistical Mechanics" (1982) — 章末 remark に未解決状況が散在。
- Jimbo, Miwa, "Algebraic Analysis of Solvable Lattice Models" (1995) — CBMS lecture, 末尾の open question 群。
- Gomez, Ruiz-Altaba, Sierra, "Quantum Groups in Two-Dimensional Physics" (1996) — 章末問題。
- Reshetikhin, "Lectures on Integrable Systems" — open problem 節あり。

## Conference proceedings / problem lists

- ICM proceedings (integrable systems sections)
- Statphys proceedings (open problems sessions)
- "Open problems in mathematical physics" 系の特集号
- MathOverflow タグ: `integrable-systems`, `quantum-groups`, `statistical-mechanics`
- arXiv "future directions" や "outlook" を含む review

## Review articles (open problem 密度が高いもの)

- Bazhanov, Lukyanov, Zamolodchikov 系の Q-operator / ODE-IM 系 review
- Foda, Wheeler 系の partition function review
- Kuniba, Nakanishi, Suzuki, "T-systems and Y-systems in integrable systems" (2011)
- Borodin, Petrov, "Integrable probability" 系の lecture notes
- Korepin, Bogoliubov, Izergin, "Quantum Inverse Scattering Method and Correlation Functions" の各章末

## Workshop / seminar notes

- Banff (BIRS) workshop reports — 未解決問題セッションを含むものが多い
- MATRIX (Australia) workshop notes
- Newton Institute programmes on integrable systems

## Open problem aggregators

- arXiv 検索: `"open problem"`, `"remains open"`, `"is an open question"`, `"we conjecture"`, `"it would be interesting to"`, `"future work"`
- Wikipedia "Unsolved problems in mathematics" の statistical mechanics 節
- Encyclopedia of Integrable Systems (Springer)

## 運用メモ

- 書籍は arXiv に source がない → metadata + 人手要約で `inputs/corpus/` に入れる
- review / workshop notes は arXiv にあれば source 優先
- aggregator はクエリ生成のヒントとして使い、本文は別途 harvest
