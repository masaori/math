## 見つかった clean 法則（Λ 側, 決定可能）

**奇数 $L$ に対し $v_2(\tau(L))=2(L-1)$**。
検証: $L=3,5,7,9,11 \Rightarrow v_2=4,8,12,16,20 = 2(L-1)$（全一致）。
（偶数 $L$ では $p=2\mid L$ で別レジーム: $v_2=19,29,61,53,83$（L=4,6,8,10,12）とより大きい。）

これは全域木数の Λ 側（$\Phi_L=\log\tau(L)$ の $\ell_2$ 係数）の**2変数 LTE 的な閉じた法則**（$c^L-1$ の1変数 LTE の2次元版に相当）。

## より複雑な部分（正直に）

- **対角 $v_p(\tau(p))$**（$L=p$ 素数）: $p=2,3,5,7,11\Rightarrow 2,6,14,14,22$。$p=3,7,11$ は $2p$（6,14,22）だが $p=5$ は 14≠10 で単純 $2p$ でない。$p$-円分体の分岐（$1-\zeta_p$ の付値）に依る **cyclotomic p 進構造**で、単純式にならない。
- 一般の $v_p(\tau(L))$（$p\nmid L$）: $p\mid\tau(L)$ となる $L$ は $4-2\cos-2\cos$ の $\bmod p$ 零点条件（＝離散ラプラシアンの $\mathbb{F}_p$ 上の核）で決まり、$\mathrm{ord}_p$ 依存。円分的で per-prime。

## 位置づけ

- **確定**: 奇 $L$ で $v_2(\tau(L))=2(L-1)$（決定可能・検証済, おそらく既知の全域木数の算術だが Λ 側 clean 法則の good example）。
- ℝ側 $(1/L^2)\log\tau\to4G/\pi$（cycle9）と対で、同一離散ラプラシアン曲線の両素点。
- 一般の $v_p$ 則は cyclotomic で複雑＝Λ 側は「単一の簡単な式」でなく円分的算術構造（cycle7 の LTE, cycle6 の p 進エントロピーと同系）。
