## 既知理論への接地（重要）

**全域木数の $\ell$ 進付値は「グラフの岩澤理論」で研究されている**:
- arXiv:2006.14012「On abelian $\ell$-towers of multigraphs」ほか（Vallières, McGown, Gonet 系）: 正則 abelian $\ell$-tower の全域木数の **$\ell$ 進付値は、数体の $\mathbb{Z}_\ell$ 拡大の類数の $\ell$ 進付値（岩澤 $\mu,\lambda$ 不変量）と平行に振る舞う**。
- ⇒ $v_p(\tau(L))$ の $L$ 依存は **グラフ版岩澤理論**の対象。これは cycle6 の「$p$ 進エントロピー＝岩澤 $\mu_p$」（Deninger, Besser–Deninger）と**同じ岩澤的構造**の別の顔。

したがって:
- **Λ 側（$\Phi_L=\log\tau(L)$ の素因数構造）の算術は、岩澤理論（グラフ/$\mathbb{Z}_\ell$ 拡大）に乗る既知の枠組み**。$v_2(\tau(L))=2(L-1)$（奇 L）はその clean な instance（$L$ が $L\times L$ の周期を動く塔での付値の線形則）。
- ℝ側（$(1/L^2)\log\tau\to4G/\pi$, cycle9）＝Mahler 測度＝エントロピー と対。両側とも既知理論（Mahler 測度 / 岩澤理論）に接続。
