# cycle 10 / T3: 自由エネルギー＝Mahler 測度＝エントロピー と Lehmer 問題

スクリプト `lehmer.sage`、出力 `.out`。文献: Smyth「Mahler measure of algebraic numbers: a survey」(math/0701397); Lehmer 1933; Boyd–Lawton; 代数力学エントロピー(arXiv:2109.08217)。

## 接続（ℝ/Λ 双対の ℝ 側が触れる famous open problem）

- ℝ/Λ 双対の **ℝ 側 = 自由エネルギー = log Mahler 測度 = 位相的/代数的エントロピー**（LSW, cycle4-9）。
- **Lehmer 問題**: $m(P)>1$ をみたす整数多項式 $P$（非円分）の Mahler 測度の**最小値**は? Lehmer 数 $\approx1.176280818$（Lehmer 多項式, Salem 数）より小さい例は未発見。$m\to1$ に近づけるか未解決。
- ⇒ **1変数（転送行列の特性多項式）の Lehmer 問題 = 「最小の正のエントロピー（自由エネルギー）」問題**。円分（1の冪根）= エントロピー 0 = 自明/frozen。
- **Boyd–Lawton**: 多変数（2D 格子 = ℤ² 作用）の Mahler 測度は 1変数の極限で、**下限のギャップなし**（エントロピー 0 に近づける）。1変数の Lehmer ギャップ（予想）とは対照的。

## 正直な注記（スケールの罠）

- **全域木エントロピー $4G/\pi=1.16624$ と Lehmer 数 $1.17628$ の近さは、スケール違いの偶然で無意味**。前者は**エントロピー（log スケール）**、後者は**Mahler 測度**（そのエントロピーは $\log1.176=0.162$）。混同しないこと。
- Mahler 測度・エントロピーは **ℝ 側（非可算・一般に非決定可能）** の量。本プロジェクトの決定可能な成果は Λ 側（p 進, cycle3-8）。Lehmer 問題は双対の ℝ 側が触れる有名 open で、**我々が解く対象ではなく「双対がどこに繋がるか」の地図**。

## T3 としての位置づけ

- **確定（概念的接続）**: 格子模型の自由エネルギー＝Mahler 測度＝エントロピー。1変数（転送行列特性多項式）の最小正エントロピー = Lehmer 問題。2D は Boyd–Lawton で下限なし。
- **open/未着手**: 物理的な転送行列の特性多項式が Lehmer 数（小エントロピー Salem 数）を実現するか。Lehmer 多項式は E8/Coxeter 等に現れるが標準格子模型との関係は非自明。
- 正直: これは ℝ 側の地図で、決定可能な Λ 側成果とは別。誇張せず「双対の ℝ 側が Lehmer に繋がる」までに留める。
