## 重要な自己訂正（cycle 4 の早合点）

- cycle 4（`onsager_structure.sage`, N=2,3 のみ）で「全固有値の最小多項式次数が2の冪＝Onsager/自由フェルミ多重2次体」と読んだが、**N=4,5 では次数 3,6,16 等の非2冪因子が出現**。
- ⇒ **スペクトルは一般には純粋な多重2次体（自由フェルミ/Onsager）ではない**。cycle 4 の「Onsager 構造」は **小 N の早合点**だった。正直に撤回する。
- 原因の可能性: この係数 $(1-\omega^{-a})^{-1}$・一般 λ は**超可積分点とは限らない**（超可積分はパラメータの特定関係が必要）。generic な chiral clock は自由フェルミでなく、混合次数の代数的スペクトルが普通。

## robust に残る事実

- **有限 N スペクトル ∈ ℚ̄・決定可能**（次数は混合でも代数的）。これは不変（T2/C-U3 の核）。
- 2次因子の√中身は部分構造として安定（λ=1/2: {33,57} が N=2..5 不変、λ=1: {3,6}）。これはスペクトルの一部であって全部ではない。

## 含意・次

- 「カイラル Potts 有限 N が Onsager 自由フェルミ構造」と主張するには **真の超可積分点**（von Gehlen–Rittenberg の具体パラメータ）で計算し直す必要（cycle 6 候補）。一般点では混合次数。
- T2 の確実な成果は「本命模型でも有限 N スペクトル∈ℚ̄・決定可能」までに留める。Onsager 構造の主張は超可積分点で再検証するまで保留。
