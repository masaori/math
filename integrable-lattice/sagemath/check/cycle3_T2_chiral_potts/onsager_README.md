# cycle 4 / T2: カイラル Potts 有限 N スペクトルの Onsager 構造

スクリプト `onsager_structure.sage`、出力 `onsager_structure.out`（SageMath 10.6, N=2,3）。

## 観察（厳密, ℚ̄）

超可積分 ℤ_3 カイラル Potts 鎖（λ 有理）の有限 N スペクトル:
- **全固有値の最小多項式次数が 2 の冪**（1,2,4）、**全実**。
- **少数の平方根で生成**（多重2次体）:
  - λ=1: 2次固有値の判別式集合 = {12,24,60} = 4·{3,6,15}（√3,√6,√15）。N とともに緩く増加。
  - λ=1/2: 判別式集合 = {33/4, 57/4}（**N=2,3 で安定**）。N=3 で次数4が出る＝splitting 体 $\mathbb{Q}(\sqrt{33},\sqrt{57})$（多重2次, 次数4）に固有値が住む。

## 読み（T2: 有限 N → 極限への橋）

- 次数が 2 の冪・スペクトルが少数の √ で生成 ＝ 超可積分カイラル Potts の **Onsager 代数／自由フェルミ構造**の有限 N での足跡（既知の構造を ℚ̄ で直接確認）。
- 準位 $E=A\pm\sum_k m_k\sqrt{\alpha_k}$ 型。$\sqrt{\alpha_k}$ の中身（判別式）の集合が N で安定（λ=1/2）/規則的に増加（λ=1）⇒ **極限のフェルミオン準位**へつながる候補データ。
- 有限 N は完全に ℚ̄ で決定可能・witness（最小多項式・判別式）付き。

## 含意・次

- T2（実際の厳密解）の足場: 有限 N の Onsager 構造（√ 集合）を N で追跡し、極限のフェルミオン分散へ外挿する道。一般カイラル Potts（高種数）でも有限 N は ℚ̄。
- 次（cycle 5+）: √ 集合 $\{\alpha_k(N)\}$ の N 依存則を定式化（Onsager 準位の代数的記述）。判別式と模型パラメータ λ の関係。
- 注: N=4 以上は QQbar 最小多項式計算が重い（背景実行が exit 144）。小 N で構造は明瞭。
