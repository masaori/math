# cycle 2 step 1 結果: v_p(Z_{N,L}) の N 依存則（データ）

スクリプト `vp_law.sage`、出力 `vp_law.out`（SageMath 10.6）。

## 観察された型（六頂点 Z_{N,L}=Tr T^N, 整数重み）

$v_p(Z_N)$（＝Massieu $\Phi_N\in\Lambda$ の $\ell_p$ 係数）は次の三型のいずれか:

1. **恒 0**（p が全 Z_N を割らない）。
2. **末尾線形**（傾き整数）。例 (1,1,2),L=2,p=2: $v_2(Z_N)=N+2$。(1,1,2),L=3,p=2: $v_2=N+3$。
3. **周期的**。例 (1,1,2),L=2,p=3: $0,1,0,1,\dots$（周期2）; p=7: 周期6。

いずれも **決定可能・ℝ 不使用**。

## 固有値による説明（Newton 多角形, step 2 で精密化）

$Z_N=\sum_i\lambda_i^N$（$\lambda_i$=T の固有値, charpoly の根）。例 (1,1,2),L=2: charpoly=$(x-6)(x+2)(x-2)^2$, 固有値 $\{6,-2,2,2\}$:
$$Z_N=6^N+(-2)^N+2\cdot 2^N.$$
- $v_2$: $6^N+(-2)^N=2^N(3^N+(-1)^N)$, $2\cdot2^N=2^{N+1}$。最小付値項が支配し $v_2(Z_N)=N+2$(線形)。
- $v_3$: $6^N=2^N3^N$ が $v_3=N$、他は $v_3=0$ ⇒ 和の $v_3$ は他項(非 3 倍)で 0/周期に。

**示唆**: $v_p(Z_N)$ は **charpoly(T) の p 進 Newton 多角形（固有値の p 進付値）** で決まり、最小付値固有値が一意なら N について線形（傾き=その付値）、付値が等しい固有値クラスタの干渉で周期/LTE 補正が出る。これは線形漸化列の p 進付値の古典理論（Z_N は charpoly の漸化を満たす）。

## 双対性（ℝ 側 / Λ 側）

- **ℝ 側**: 自由エネルギー $\sim\log\lambda_{\max}$（Perron 固有値, 絶対値最大）。極限・解析。
- **Λ 側**: $\Phi_N$ の素因数構造 $v_p(Z_N)$ は **p 進付値最小の固有値**（Newton 多角形）で決まる。可算・決定可能。
- 同じ固有値集合の「絶対値」と「p 進付値」が、それぞれ ℝ 側の自由エネルギーと Λ 側の Φ 数論構造を支配する、という分担。
