# cycle 2 step 2 結果: v_p(Z_N) の固有値 p 進付値による説明

スクリプト `eigenvalue_link.sage`、出力 `eigenvalue_link.out`（SageMath 10.6）。

## 予想と検証

**予想**: $v_p(Z_{N,L}) = \mu_{\min}(p)\cdot N + r_p(N)$。
- $\mu_{\min}(p)$ = charpoly(T) の **p 進 Newton 多角形の最小傾き**（＝固有値の最小 p 進付値）。
- $r_p(N)$ = 残差（最終周期的）。

**機構**: $Z_N=\sum_i\lambda_i^N$。最小付値 $\mu_{\min}$ を括ると $Z_N=p^{\mu_{\min}N}\sum_i u_i^N$（$u_i=\lambda_i/p^{\mu_{\min}}$ は付値≥0、最小付値群は p 進単位）。単位和 $\sum u_i^N$ は単位係数の線形漸化列で、mod $p^k$ が周期的 ⇒ その $v_p=r_p$ は最終周期的。

**検証結果**（六頂点, p∈{2,3,5,7}）:
- 大半で予想成立（残差が最終周期）。例: (1,1,2)L2 p=2: $\mu_{\min}=1$, 残差 ≡ 2（周期1）⇒ $v_2(Z_N)=N+2$。(1,1,2)L2 p=3: $\mu_{\min}=0$, 残差 周期2。
- **1例で周期検出失敗**: (1,1,1)L2, p=7（固有値 1,2,2,3）。$Z_N=1+3^N+2^{N+1}$, mod7 は周期6（$N\equiv5\pmod6$ でのみ $7\mid Z_N$）。残差は本質的に周期6だが、零剰余指数で稀に深い付値スパイク（$v_7\ge2$）が出うる。

## 正直な評価（数論的位置づけ）

- この「$v_p$ = 線形 + 最終周期」は **線形漸化列の p 進付値の理論**そのもの。$Z_N=\mathrm{Tr}\,T^N$ は charpoly の漸化を満たす整数列で、その p 進付値は **eventually periodic だが、固有値部分和が p 進的に消える稀な指数（Skolem–Mahler–Lech 現象）で例外スパイクを持ちうる**。これは既知の深い微妙さ。
- したがって D-U2 は「全く新しい数論」ではなく、**既知の p 進線形漸化理論を可積分転送行列の Massieu Φ に適用した具体的構造定理**。新規性は (a) 可積分模型の Φ∈Λ への適用と (b) ℝ側(log λ_max)/Λ側(最小付値固有値)の双対の明示にある。
- 厳密定理化には SML 例外の扱いが要る（step 4 で caveat 付き命題に）。

## ステータス

予想は generically 成立。例外（SML スパイク）を含む正確な命題は step 4（theorem_candidate）で caveat 付きで明文化。
