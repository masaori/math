# cycle 2 step 3(従)結果: 3-状態 Potts での v_p 則の普遍性

スクリプト `potts_phi_vp.sage`、出力 `.out`（SageMath 10.6）。

## 確認できたこと

別の可積分族（3-状態 Potts, 整数重み行転送行列）でも:
- $Z_{N,L}=\mathrm{Tr}\,T^N\in\mathbb{Z}_{>0}$、$\Phi_N=\log Z_N\in\Lambda$（素因数分解）。決定可能・ℝ 不使用。
- $v_p$ 則 $v_p(Z_N)=\mu_{\min}(p)N+r_p(N)$ は **多くの素数で成立**（例 q=3,L=3,x=2,p=2: $v_2$ ≡ 一定 ⇒ 線形）。

## 正直な観察（特殊素数 p|q）

- **$p=q=3$（状態数に等しい素数）で残差が単純周期に乗らない**ように見えた（検出器が None）。が、手で見ると最終的には周期的:
  例 q=3,L=2,x=2,p=3: $v_3=[2,6,2,2,5,2,2,5,2,2]$。N=2 の 6 は早期スパイク、N≥4 は周期3 $[2,5,2]$。つまり**最終周期＋早期 SML スパイク**で、素朴検出器が過小報告しただけ。
- 一般に: $v_p$ は **eventually periodic + 線形**だが、(i) 早期遷移スパイク、(ii) 特殊素数 $p\mid q$（模型の対称性 $S_q$ の位数）で周期が大きく・スパイク頻、という構造。これは p 進線形漸化列の既知の挙動（SML 領域）と整合。

## 含意

- D-U2 の構造（Φ∈Λ の ℓ_p 係数 = μ_min·N + 最終周期, SML 例外つき）は **六頂点固有でなく、整数転送行列をもつ可積分族に普遍**（Potts でも成立）。
- ただし**正確な定理化には (a) SML 早期スパイク (b) 特殊素数 p|q の周期増大** を caveat として要する。これは step 4 の定理候補に反映。
