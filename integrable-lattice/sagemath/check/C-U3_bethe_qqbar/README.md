## 確認できたこと（厳密）

相互作用可積分模型 XXZ 鎖（六頂点の対数微分。熱力学極限の相関は未解決母集団）について、有理異方性 Δ∈{1/2, 2, −1/3}（ℝ 不使用）・N=2,3,4 で:

- 特性多項式は ℚ[x]（Δ 整数なら ℤ[x]）。
- 全固有値が **QQbar（代数的数）に住む**（`all in QQbar = True`）。
- 各固有値は **ℚ̄ 上の最小多項式を witness** にもつ（小 N では次数 1〜2）。等号・順序は根分離で決定可能。
- 例 N=3, Δ=1/2: charpoly = (x−7/2)²(x−3/2)²(x+5/2)⁴。

## 解釈（正直に）

- **主張「有限 N スペクトル ∈ ℚ̄・決定可能」は厳密に成立**し、ℝ を一切使わずに最小多項式 witness で述べられる。
- ただし小 N・低異方性では固有値次数は低い（1〜2）。「数が複雑」が要点ではなく、**構造的分離**が要点: 有限 N は完全に代数的・決定可能、ℝ 脱出は N→∞ の極限（相関の閉形式＝未解決）側にのみ生じる。
- Ising（既解, 自明）と違い、XXZ は**極限が未解決の相互作用模型**。そこで「有限 N＝可算決定可能 vs 極限未解決」の分離が非自明な内容をもつ（本プロジェクト本命母集団の最小実例）。
