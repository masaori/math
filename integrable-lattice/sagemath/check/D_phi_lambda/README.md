# D 検証結果: 六頂点 Massieu Φ_N ∈ Λ の素因数構造

スクリプト `sixvertex_phi_lambda.sage`、出力 `sixvertex_phi_lambda.out`（SageMath 10.6）。

## 確認できたこと（厳密）

六頂点模型（可積分。熱力学極限の相関は未解決母集団）を R 行列モノドロミーから転送行列 T（整数重み ⇒ 整数行列）として構成し、N×L トーラスの $Z_{N,L}=\mathrm{Tr}\,T^N$ を計算:

- 全 $Z_{N,L}\in\mathbb{Z}_{>0}$（数え上げの整数）。T の固有値は全て QQbar。
- $\Phi_N=\log Z_N\in\Lambda$（素因数分解の指数ベクトル）として厳密に得られる。等号・順序は素因数分解・整数比較で決定可能、ℝ 不使用。

## 非自明な数論的構造（D-U2 の実データ）

- **(a,b,c)=(1,1,2), L=2**: $Z_N=8,48,224,1344,7808,46848$。**2進付値 $v_2(Z_N)=N+2$（線形成長）**、奇部 $1,3,7,21,61,183$。Φ_N の $\ell_2$ 係数が N の一次式。
- **(1,1,1) [ice point], L=2**: $Z_N=8,18,44,114,308,858$。$Z_N=\sum_i\lambda_i^N$ の整数線形漸化（D-U1）に従う整数列で、Φ_N の素因数は N とともに sporadic な大素数（11,19,7·11,…）を含む。
- 一般に $Z_N=\sum\lambda_i^N$（$\lambda_i\in$ QQbar）⇒ Φ の N 依存は代数的固有値が支配（D-U1）。整数線形漸化が Λ の素因数構造に効く。

## 解釈

- **Φ_N ∈ Λ は厳密・決定可能**で、しかも $p$ 進付値の N 依存（D-U2）という**非自明な数論的内容**を持つ。これは「整数多項式の根は代数的」(A の自明さ)とは違い、観察に値する構造。
- 六頂点の極限（相関の閉形式）は未解決母集団。だが**有限 N の Massieu Φ は Λ で完全に閉じ**、その数論的構造は計算可能。これが本プロジェクトの本命（有限 N＝可算決定可能 vs 極限未解決）の具体例。

- 本命模型（カイラル Potts τ^(2) / 高ランク）への適用は step 4。
