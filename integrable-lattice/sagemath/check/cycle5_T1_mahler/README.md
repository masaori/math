## 設定（代数力学 Lind–Schmidt–Ward の枠）

$P(z,w)=5-(z+z^{-1})-(w+w^{-1})\in\mathbb{Z}[z^{\pm},w^{\pm}]$（トーラス上零点なし、最小値 1）。

## 意義

- ℝ/Λ 双対の**最小・厳密な実証**: 有限 $L$ は $\mathbb{Z}/\Lambda$ で完全決定可能、極限 $m(P)$ のみが $\mathbb{R}$。
- ℝ側（成長率＝Mahler 測度）は LSW 定理で**厳密**（Ising/dimer で精密: arXiv:2407.19531）。
- Λ側（$a_L$ の $p$ 進構造）は D-U2 命題 A（$v_p$ の周期性, 決定可能）が適用。$p$ 進 Mahler 測度との厳密同定は予想（Besser–Deninger 系, cycle 6+ 文献精査）。
- これで研究ノート `docs/research/R-Lambda-duality/` の双対が具体例で裏付けられた（同一 $P$, 両素点を実計算）。
