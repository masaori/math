# cycle 9 / T1: 2変数スペクトル曲線（全域木/ダイマー/GFF）の両素点

スクリプト `spanning_tree_both_places.sage`、出力 `.out`（SageMath 10.6）。

## モデル（実際の格子模型のスペクトル曲線）

$P(z,w)=4-(z+z^{-1})-(w+w^{-1})\in\mathbb{Z}[z^{\pm},w^{\pm}]$ ＝ 離散ラプラシアン。全域木・ダイマー・ガウス自由場の曲線。
matrix-tree 定理: $L\times L$ トーラスの全域木数
$$\tau(L)=\frac1{L^2}\prod_{(j,k)\ne(0,0)}\Bigl(4-2\cos\tfrac{2\pi j}{L}-2\cos\tfrac{2\pi k}{L}\Bigr)\in\mathbb{Z}.$$
（$a_L=\prod_{(z,w)\ne(1,1)}P=L^2\tau(L)$。$P$ は $(1,1)$ に torus 零点をもつので $(1,1)$ を除く。）

## 両素点（確認済み）

- **ℝ 側**: $(1/L^2)\log\tau(L)\to 4G/\pi=1.16624$（$G$=Catalan 定数）＝**全域木エントロピー（既知）**。単調増加で収束を確認。
- **Λ 側**: 同じ $\tau(L)\in\mathbb{Z}$ の素因数分解 = $\Phi_L\in\Lambda$（決定可能）。

## 意義

- **cycle5 の toy $P$ を、実際の格子模型（全域木/ダイマー/GFF）の 2変数スペクトル曲線へ拡張**。
- **ℝ 側が既知の物理定数 $4G/\pi$ と一致**＝この枠組み（同一整数曲線の Mahler 測度＝自由エネルギー）の正しさの裏付け（Reframe の検証）。
- Λ 側の素因数構造に規則性: $v_5(\tau(5))=14,\ v_7(\tau(7))=14$（素数 $=L$ で高冪）。円分/LTE 的構造（cycle7 の $c^L-1$ と同種、2変数版）。
- 双対: 同一 $P$ から ℝ（Catalan 定数）と Λ（素因数構造）。有限 $L$ は $\mathbb{Z}/\Lambda$ 決定可能、極限 $4G/\pi$ が $\mathbb{R}$。
