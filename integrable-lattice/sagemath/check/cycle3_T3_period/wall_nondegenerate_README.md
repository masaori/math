## 統計的に正直な評価（重要）

- 一般側の破れ基準率 ≈ 2.1%。**この基準率なら 43 件中 0 件は偶然でも約 $0.979^{43}\approx0.40$ の確率で起こる**。
- ⇒ **六頂点 0/43 は統計的に有意でない**。「可積分性が Wall 等式を保護する」とは**結論できない**（偶然と区別不能）。
- 有意差を見るには六頂点側を数百〜千件規模で検査する必要（より大きい L・重み範囲・p 範囲）。それは cycle 6+ の課題。

## 確定している事実

1. **Wall 等式は非退化でも一般には成立しない**（Pell p=13 等, 確定）。rigorous 上界 $\pi(p,k)\mid p^{k-1}\pi(p,1)$ は不変。
2. 六頂点で破れは観測されていないが、標本不足で可積分性の効果は**未確定**。
3. D-U2 命題 A（周期の存在・決定可能性）は Wall とは独立に成立（不変）。
